## 1. 🤔 왜 이 분석을 해야 하는가

### 교육 담당자가 겪는 현실적인 문제

지금까지 교육을 마치면 **"잘 된 것 같다"** 또는 **"아쉬웠다"**라는 감(感)만 남았어요.
하지만 이런 질문에 숫자로 대답할 수 없었어요:

- 시험 성적이 좋았던 교육생이 매장에서도 잘 팔까?
- 교육일지를 성실히 쓴 교육생이 실전에서도 성장이 빠를까?
- 4주 교육 중 어떤 주차의 내용이 실제 수주에 가장 도움이 되었을까?
- 2기 교육을 설계할 때, 어디에 시간을 더 투자해야 할까?

### 이 기획서의 목표

1기는 자동화 시스템(Supabase에 데이터가 자동으로 쌓이는 것)을 도입한 **최초의 기수**예요.
이 데이터를 활용해서:

| 목표 | 설명 |
|------|------|
| 🔗 **연결고리 찾기** | 교육 과정과 실전 매출 사이의 관계를 눈에 보이게 만들어요 |
| ⚠️ **조기 경보 만들기** | 교육 중에 위험 신호를 잡아서 빠르게 도와줄 수 있게 해요 |
| 📋 **2기 근거 만들기** | "느낌"이 아닌 "데이터"로 다음 교육을 설계해요 |

> 💡 1기는 비교 대상이 없는 첫 기수이므로, **교육생 간 상대 비교**와 **시간에 따른 변화 추이**에 집중해요.

## 2. 🔄 3단계 추적 프레임워크 (Framework — 전체 구조를 한눈에 보는 틀)

교육생 한 명의 여정을 3단계로 나누어 추적해요.
각 단계의 데이터가 다음 단계를 **예측**하는 데 쓰여요.

```
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   🎓 1단계       │      │   🏪 2단계       │      │   💰 3단계       │
│   입문교육       │ ───→ │   심화교육       │ ───→ │   매장 실전      │
│   (~4/22)        │      │   (4/23~5/22)    │      │   (5/23~)        │
│                  │      │                  │      │                  │
│  시험 + 교육일지  │      │  R&P + 실습일지   │      │  실제 수주 금액   │
│  출결 + 질문하기  │      │  매장 적응        │      │  매출 기여        │
└─────────────────┘      └─────────────────┘      └─────────────────┘
      📚 학습력                 🔧 적용력                📈 성과력
```

### 핵심 연결 질문 — "이것이 저것에 영향을 줄까?"

| 연결 | 검증하고 싶은 질문 | 왜 중요한가 |
|------|-------------------|------------|
| 1단계 → 2단계 | 시험 잘 본 사람이 R&P(역할극 — 실제 상담 연습)도 잘하나? | 지식이 실전 연습으로 이어지는지 확인 |
| 1단계 → 3단계 | 입문교육 성적이 수주 실적을 예측하나? | **교육 자체의 가치를 증명**하는 가장 중요한 질문 |
| 2단계 → 3단계 | R&P 평가와 수주 실적은 비례하나? | 심화교육 훈련이 실전에서 통하는지 확인 |
| 1단계 태도 → 3단계 | 성실한 태도(출결, 일지)가 매장 성과와 관련 있나? | 지식뿐 아니라 태도도 성과를 만드는지 확인 |

## 3. 📏 각 단계별 핵심 지표(KPI — 핵심성과지표, 꼭 봐야 할 숫자들)

### 3-1. 🎓 1단계: 입문교육 지표 (이미 Supabase에 자동 수집 중)

#### A. 지식 습득 지표 — "얼마나 알게 되었나?"

| 지표 | 어디서 나오나 | 어떻게 계산하나 | 💡 왜 중요한가 |
|------|-------------|---------------|---------------|
| **차시별 평균 점수** | test_scores (시험 점수 테이블) | 교육생별 전체 차시 점수의 평균 | 전반적인 지식 수준을 한 눈에 보여줘요 |
| **성장 기울기** | test_scores | 처음 3회 평균 vs 마지막 3회 평균의 차이 | 점수가 높아도 성장이 없으면 정체, 점수가 낮아도 급성장 중이면 가능성 있어요 |
| **대분류별 정답률** | test_responses + questions (문항별 응답 + 문제은행) | 대분류 9개(학생방/침실/옷장/키즈룸/거실/주방/서재/조명/공통지식) 각각의 맞힌 비율 | 어떤 제품군 지식이 약한지 → 심화교육 때 보강 가능 |
| **소분류별 정답률** | test_responses + questions | 소분류 14개(제품규격/소재공법/제품특장점/색상옵션/기능구조 등)별 맞힌 비율 | "제품 크기는 아는데 색상을 모른다" 같은 **지식 유형별 약점** 발견 |
| **만성 오답 비율** | coaching_reports (AI 코칭 리포트)의 tag_tracking | chronic(고질 약점) 태그 수 / 전체 약점 수 | 반복해서 틀리는 문제가 많다 = 학습법 자체를 바꿔야 한다는 신호 |

#### B. 태도/참여 지표 — "얼마나 열심히 했나?"

| 지표 | 어디서 나오나 | 어떻게 계산하나 | 💡 왜 중요한가 |
|------|-------------|---------------|---------------|
| **출석률** | attendance (출결 테이블) | (출석일수 / 전체 교육일수) × 100 | 기본적인 성실성. 결석/지각이 잦으면 매장에서도 근태 문제 가능성 |
| **교육일지 제출률** | student_notes (교육일지 테이블) | 제출일수 / 전체 교육일수 | 일지를 매일 쓰는 사람은 하루를 정리하는 습관이 있어요 |
| **자신감 추이** | student_notes의 meta.confidence | confident/half/low를 5/3/1로 점수화하여 추이 관찰 | 자신감이 올라가는 사람 vs 계속 낮은 사람 → 심리적 지원 필요 여부 판단 |
| **참여 점수** | student_notes의 participation_score | STEP 1~3 작성 여부 (0~3점) | 일지를 제출은 했지만 성의 없이 쓴 건지 구별 |
| **자율학습 노트 수** | student_notes ('자율학습' 태그) | 개수 카운트 | 자기주도 학습 의지의 직접적 지표 |
| **질문 빈도** | student_questions (질문하기 테이블) | 질문 개수 | 질문을 많이 하는 교육생이 보통 더 빨리 성장해요 |
| **교육자 메모 패턴** | student_memos (교육자 메모 테이블) | 카테고리별 빈도 (칭찬 vs 주의) | 교육자가 관찰한 정성적 평가를 숫자로 바꾼 것 |
| **실습일지 실적 추이** | student_notes (실습일지 태그) | 상담건수/견적건수/수주건수/수주금액의 주간 변화 | 입문교육 중 매장 실습 성과가 늘고 있는지 |

#### C. 🎯 1단계 종합 점수: 입문교육 적응 지수

위 지표들을 종합해서 교육생별로 **"입문교육 적응 지수"**를 만들어요.
이 점수 하나로 "이 교육생이 입문교육에서 얼마나 잘 적응했는가?"를 볼 수 있어요.

##### 가중치 (v3 — 2026-04-12 재설계, 8개 항목)

| 항목 | 가중치 | 무엇을 보나요 |
|------|-------|-------------|
| 📝 시험 평균 | 30% | 전체 차시 시험 점수의 평균 |
| 🗺️ 하위 분야 | 15% | 60점 미만인 제품 카테고리 개수 (약점 이름까지 노출) |
| 📋 출석률 | 13% | 출석/지각/결석 (지각은 0.5로 계산) |
| 📓 교육일지 참여 | 15% | 일지 제출과 STEP 작성 충실도 (실습일지 가중치 흡수) |
| 📈 성장 기울기 | 10% | 최근 3차시와 초반 3차시 점수 비교 |
| 💪 자신감 추이 | 8% | 일지 자신감 단계의 변화 (상승/유지/하락) |
| 🔁 만성 오답 | 5% | 계속 반복해서 틀리는 문항이 얼마나 되는지 |
| 🗒️ 메모 톤 | 4% | 교육자 칭찬/주의 메모의 비율 |

##### ⚠️ 부정 신호 델타 감점 (최대 −8점)

점수가 괜찮아도 "조용히 무너지는" 학생을 놓치지 않기 위한 장치. 총점에서 직접 차감.

- 최근 시험 점수가 **초반보다 5점 넘게 떨어짐** → −5점
- 일지 자신감이 **계속 낮은 상태로 유지** → −3점

##### 그룹 분류

- **상**: 75점 이상
- **중**: 50~74점
- **하**: 50점 미만

> 📝 **변경 이력**
> - **v1 (초기)**: 시험 평균(40%) + 하위 분야(20%) + 출석률(15%) + 교육일지(15%) + 자신감(10%)
> - **v2 (1기 운영 중, 2026-04-11)**: 성장 기울기/만성 오답/메모 톤/실습일지 추가 → 9개 항목
> - **v3 (2026-04-12)**: 실습일지를 교육일지 참여로 흡수 → 8개 항목. 부정 신호 델타 감점 도입
> - **자신감 미입력 처리**: v1은 중립 50점 → v3부터는 가중치 8%를 시험(+4%)과 교육일지 참여(+4%)로 자동 분산
> - ~~질문 빈도~~ → **제거**: 실제 질문이 아닌 것도 포함될 수 있어 정확도 낮음
> - 가중치는 1기 결과를 보고 **2기 때 실제 수주와의 상관도를 봐서 재조정할 예정**

#### D. 🧑‍💼 주의 교육생 판정 및 HR 조언 자동 생성 (2026-04-12 추가)

적응 지수와는 별도로 **"지금 바로 개입해야 하는 학생"**을 판정하는 체크리스트.

##### 6가지 주의 신호 체크

| 체크 항목 | 해당 조건 |
|---------|---------|
| ⚠️ 시험 평균 | 전체 차시 평균이 60점 미만 |
| ⚠️ 하위 분야 | 60점 미만인 제품 카테고리가 4개 이상 |
| ⚠️ 출석률 | 출석률이 80% 미만 |
| ⚠️ 자신감 | 최근 일지 자신감이 3회 연속 낮음 |
| ⚠️ 주의 메모 | 교육자 주의 메모가 2건 이상 |
| ⚠️ 최근 하락 신호 | 성적 5점↓ 하락 또는 자신감 급락 |

하나라도 해당되면 "주의 교육생"으로 표시. 신호 3개 이상 동시 해당 시 "위험"으로 강조.

##### 8가지 유형 자동 판정 + 맞춤 HR 조언

트리거된 체크 조합을 보고 8가지 유형 중 하나로 분류. 각 유형마다 "어려움 서술 + 권장 액션 2~5개"가 자동 생성되며, 실제 약점 카테고리 이름/하락폭/자신감 수치 등이 문장에 꽂혀서 구체적으로 나옴.

| 유형 | 판정 조건 | 핵심 접근 |
|------|---------|---------|
| 🚨 복합 위기형 | 주의 신호 3개 이상 동시 | 긴급 면담 + HR 공유 + 주 2회 체크인 |
| 📘 지식 부족형 | 시험 낮음 + 하위 분야 많음 | 1:1 튜터링 + 학습법 진단 |
| 🎯 약점 편중형 | 시험은 양호, 하위 분야만 많음 | 해당 제품군 집중 보강 |
| 💔 심리 위축형 | 자신감 계속 낮음 | 개인 면담 + 작은 성공 경험 |
| 🌊 하락 징후형 | 성적이나 자신감 최근 하락 중 | 가벼운 티타임 면담 + 번아웃 체크 |
| 🚪 근태/동기 이슈형 | 출석률만 낮음 | 비공식 사정 파악 + 개선 목표 |
| 🔍 행동 관찰형 | 주의 메모만 쌓임 | 패턴 분석 + 구체 피드백 |
| 💎 부분 주의형 | 적응 '상' 그룹인데 한두 가지 주의 | 강점 유지 + 약점만 가볍게 |

> 💡 **왜 중요한가**: "잘하던 애가 무너지는" 사각지대를 잡기 위해 🌊 하락 징후형을 따로 두었어요. 적응 점수가 '상'이어도 최근 -5점 넘게 하락했거나 자신감이 급락했다면 별도 케어 대상으로 표시돼요.

### 3-2. 🏪 2단계: 심화교육 지표 (4/23부터 수집)

> ⚠️ **참고**: 실습일지(상담건수/수주건수 등)는 입문교육 기간에 작성하는 것이므로 **1단계에 포함**돼요.
> 심화교육(매장배치 후)에서는 아래 4가지를 중심으로 봐요.

| 지표 | 어디서 나오나 | 어떻게 계산하나 | 💡 왜 중요한가 |
|------|-------------|---------------|---------------|
| **심화교육 시험 점수** | 관리자가 출제하는 테스트 | 시험별 점수 + 추이 | 매장 배치 후에도 지식이 유지/성장하는지 |
| **R&P 평가 주차별 변화** | weekly_evaluations (주차별 평가 테이블) | 강점 태그 수 변화, 개선 태그 수 변화 | 실제 상담 연습(롤플레잉)에서 실력이 늘고 있는지 |
| **강점 태그 TOP 3** | weekly_evaluations의 strength_tags | 태그별 횟수 세기 | 이 교육생의 무기가 뭔지 한눈에 파악 |
| **개선 태그 TOP 3** | weekly_evaluations의 improvement_tags | 태그별 횟수 세기 | 반복되는 약점 = 독립 후에도 주의해야 할 포인트 |
| **R&P 영역 커버리지** | weekly_evaluations의 rp_area | 10개 영역(학생방/다이닝/소파 등) 중 몇 개를 경험했는지 | 다양한 상황 경험 = 현장 적응력 |
| **벤치마킹 실행도** | benchmarks (벤치마킹 테이블) | 작성 여부 + action_plan(실행 계획) 작성 여부 | 선배를 관찰하고 배우려는 자세 |
| **최종 총평 점수** | final_evaluations (교육 총평 테이블) | overall_rating(1~5) + 4개 세부 점수 | 교육 전체를 관통하는 매장 관리자의 종합 판단 |

#### 최종 총평 세부 점수 (매장 관리자가 1~5점으로 평가)

| 세부 점수 | 의미 | 💡 왜 보는가 |
|----------|------|------------|
| store_fit_score (매장 적합도) | 해당 매장 분위기/고객층과 맞는지 | 배치가 잘 됐는지 확인 |
| independence_score (독립 수행) | 혼자 상담할 수 있는 수준인지 | 독립 근무 준비 상태 |
| customer_score (고객 응대) | 고객과의 소통 능력 | 서비스 품질 |
| product_score (제품 지식) | 현장에서 제품 설명이 가능한지 | 입문교육 지식이 현장에서 통하는지 |

### 3-3. 💰 3단계: 매장 실전 지표 (수지님이 매주 수동 전달)

| 지표 | 어디서 나오나 | 어떻게 계산하나 | 💡 왜 중요한가 |
|------|-------------|---------------|---------------|
| **주간 상담 건수** | 수주 데이터 (수지님 전달) | 그 주에 상담한 고객 수 | 고객 접점(만남) 자체가 있는지 — 만남이 없으면 수주도 없어요 |
| **주간 수주 건수** | 수주 데이터 | 계약까지 이어진 건수 | 상담을 수주로 바꾸는 능력 |
| **수주 분야** | 수주 데이터 (비고란) | 학생방/침실/거실/키즈룸/서재 등 어떤 분야 제품을 팔았는지 | ⭐ **입문교육 성적과 직접 비교 가능!** 리빙 분야 수주가 많으면 → 리빙 관련 시험 성적과 비교 |
| **주간 수주 금액** | 수주 데이터 | 총 수주 금액 | 비싼 상품을 파는지 (업셀링 능력 — 더 좋은 상품 추천하는 것) |
| **상담→수주 전환율** | 계산 | (수주 건수 / 상담 건수) × 100 | ⭐ **가장 중요한 실전 지표**. 상담을 얼마나 효율적으로 매출로 바꾸는지 |
| **주간 수주 금액 성장률** | 계산 | (이번주 - 지난주) / 지난주 × 100 | 정체인지 성장인지 한눈에 판별 |

## 4. 🔀 교차 분석 항목 — "A를 잘하면 B도 잘할까?"

교차 분석은 두 가지 데이터를 엮어서 **"이것과 저것 사이에 관계가 있나?"**를 확인하는 작업이에요.
1기 데이터로 아래 분석을 수행하면, **2기 교육 설계의 핵심 근거**가 돼요.

### 4-1. 📚 입문교육 성적 × 💰 수주 실적

| 분석 항목 | 방법 | 💡 기대하는 인사이트 |
|----------|------|-------------------|
| 시험 평균 점수 vs 전환율 | 교육생을 시험 점수 상/중/하 그룹으로 나눠서 각 그룹의 평균 전환율 비교 | "시험 잘 보면 수주도 잘하나?" → 만약 상관없다면 **시험 방식을 바꿔야** 해요 |
| 카테고리별 정답률 vs 수주 분야 | 높은 정답률 카테고리와 실제 수주 분야 매칭. 예: 리빙 수주 많은 교육생 → 거실/소파 시험 정답률 비교 | **"잘 아는 분야를 잘 파는지"** → 약한 분야를 보강하면 해당 분야 매출도 올라갈 수 있다는 근거 |
| 성장 기울기 vs 수주 성장률 | 시험 성장이 빠른 사람이 수주 성장도 빠른지 비교 | 초기 점수보다 **"성장 속도"**가 중요한 예측 변수인지 확인 |

### 4-2. 🙋 태도/참여 × 💰 수주 실적

| 분석 항목 | 방법 | 💡 기대하는 인사이트 |
|----------|------|-------------------|
| 출석률 vs 수주 실적 | 출석률 상위/하위 그룹별 수주 비교 | 근태 성실성이 매출과 연결되는지 |
| 교육일지 참여 점수 vs 수주 실적 | 일지 성실도 상위/하위 그룹 비교 | **성찰 습관**(하루를 돌아보는 것)이 현장 성과에 미치는 영향 |
| 자율학습 노트 수 vs 수주 실적 | 자율학습 횟수별 수주 비교 | 자기주도 학습이 실전 성과를 만드는지 |
| 질문 빈도 vs 수주 전환율 | 질문 많이 한 사람의 전환율 비교 | 적극성이 상담 전환으로 이어지는지 |

### 4-3. 🏪 심화교육 평가 × 💰 수주 실적

| 분석 항목 | 방법 | 💡 기대하는 인사이트 |
|----------|------|-------------------|
| R&P 강점 태그 vs 수주 | 특정 강점 태그(예: '업셀링 적극적')를 받은 교육생의 수주 금액 비교 | **어떤 역량이 실제 매출을 만드는지** |
| R&P 개선 태그 vs 수주 | '자신감 부족' 태그를 받은 교육생의 전환율 | 심화교육에서 발견된 약점이 실전에서도 약점인지 |
| 최종 총평(1~5점) vs 수주 | 총평 점수 그룹별 수주 평균 비교 | 관리자의 "감"이 실제 성과와 일치하는지 |
| 실습일지 수주 → 실전 수주 | 실습 기간 수주와 독립 후 수주 비교 | 선배 옆에서의 성과가 독립 후에도 유지되는지 |

### 4-4. ⭐ 종합 교차 분석 (가장 중요!)

| 분석 | 설명 | 💡 왜 가장 중요한가 |
|------|------|-------------------|
| **입문교육 적응 지수 vs 수주 전환율** | 종합 점수가 실전 성과를 얼마나 잘 예측하는지 | 2기에서 **"위험 교육생"을 조기 발견**하는 기준으로 사용 |
| **교육자 메모(정성) vs 수주(정량)** | "칭찬" 메모가 많은 교육생 vs "주의" 메모가 많은 교육생의 수주 차이 | 교육자의 관찰이 얼마나 정확한지 검증 |
| **만성 오답 카테고리 vs 수주 약점** | 입문교육에서 끝까지 못 외운 분야가 매장에서 놓치는 상품군과 일치하는지 | "이걸 못 외우면 이 상품을 못 판다"는 직접적 연결 |

**포함 내용:**

1. **교육생별 성적표** — 차시별 점수 + 전체 평균 + 성장 기울기

```
대시보드 (실시간, 항시 확인)
    │
    │  특정 시점에 스냅샷 찍기
    ▼
리포트 (PDF, 인쇄/공유)
```

- 대시보드의 데이터를 **특정 시점에 PDF로 내보내면** 그게 리포트가 돼요
- 대시보드는 "지금"을 보는 것, 리포트는 "그때"를 기록하는 것
- 대시보드에서 **📄 PDF 내보내기 버튼**을 누르면 리포트가 자동 생성되는 구조

## 7. 🎯 2기 교육 설계를 위한 5가지 최종 산출물

1기의 모든 분석이 끝나면, 아래 5가지를 만들어요.
이것이 **2기 교육을 바꾸는 구체적인 근거**가 돼요.

### 산출물 1: 📊 예측 변수 랭킹표

**"어떤 교육 지표가 매장 성과를 가장 잘 예측하는가?"**

| 순위 | 입문교육 지표 | 수주 전환율과의 상관도 | 해석 |
|------|-------------|---------------------|------|
| 1 | _(1기 데이터로 채울 예정)_ | 강함/보통/약함 | ... |
| 2 | | | |
| 3 | | | |

> **활용**: 상관도가 높은 항목에 2기 교육 시간을 더 배분해요.
> 예를 들어 "자율학습 노트 수"가 수주와 가장 상관이 높다면, 2기에서는 자율학습 시간을 늘려요.

### 산출물 2: 📋 카테고리별 교육 효과 분석표

| 대분류 | 평균 정답률 | 해당 분야 수주 비중 | 교육 시간 대비 효율 | 2기 시간 조정 제안 |
|--------|-----------|-------------------|-------------------|------------------|
| 학생방 | _(데이터)_ | _(데이터)_ | 높음/보통/낮음 | 유지/확대/축소 |
| 침실 | | | | |
| ... | | | | |

> **활용**: "침실은 시험 정답률이 높은데 수주가 적다면 교육 시간을 줄이고, 거실은 정답률이 낮은데 수주가 많다면 시간을 늘려야 한다"

### 산출물 3: ⚠️ 위험 교육생 조기 발견 체크리스트

1기 데이터에서 **"결국 수주가 부진했던 교육생"**의 입문교육 시점 특징을 역추적해요.

```
[ ] 입문교육 시험 평균 __점 이하
[ ] 성장 기울기 마이너스 (점수가 오히려 떨어지는 추세)
[ ] 출석률 __% 이하
[ ] 교육일지 자신감 3회 연속 'low'
[ ] 교육자 메모 '주의' 카테고리 __건 이상
[ ] 질문 0건
[ ] 자율학습 노트 0건
```

> **활용**: 2기 교육 2주차부터 이 체크리스트로 위험 교육생을 빠르게 발견 → 즉시 면담/보강

### 산출물 4: 🏷️ R&P 태그 실효성 분석표

| 강점 태그 | 보유 교육생 수 | 보유자 평균 수주 | 미보유자 평균 수주 | 태그 실효성 |
|----------|-------------|----------------|-----------------|-----------|
| 고객 라포형성 우수 | _(데이터)_ | _(데이터)_ | _(데이터)_ | 높음/낮음 |
| 업셀링 적극적 | | | | |
| ... | | | | |

> **활용**: 실효성이 높은 태그 = 2기 심화교육에서 더 집중 훈련할 역량
> 실효성이 낮은 태그 = 태그 기준 자체를 다시 생각해봐야 해요

### 산출물 5: 📑 1기 교육 종합 보고서

위 1~4번을 합쳐서 **경영진/인사팀에 보고**할 수 있는 문서:

**1장 요약 (경영진용):**
- 1기 교육생 N명 중 N명 수료, N명 퇴사
- 평균 수주 전환율 N%, 평균 월 수주 금액 N만원
- 교육 투자 대비 매출 기여 효과
- 2기 교육 개선 방향 3가지

**상세 본문 (교육 담당자용):**
- 전체 교차 분석 결과
- 교육생 개인별 3단계 통합 프로필
- 교육 커리큘럼 개선안 (시간 배분, 콘텐츠 변경, 평가 방식 조정)

## 📎 부록: 데이터 테이블 맵

분석에 사용하는 데이터 저장소(Supabase 테이블)와 각 분석에서의 역할을 정리해요.

| 테이블 | 단계 | 분석 용도 |
|--------|------|----------|
| students | 전체 | 교육생 기본 정보, 매장 위치, 퇴사 여부 |
| test_scores | 1단계 | 차시별 점수, 평균, 성장 기울기 |
| test_responses + questions | 1단계 | 카테고리/소분류별 정답률, 만성 오답 |
| attendance | 1단계 | 출석률, 근태 패턴 |
| student_notes (교육일지) | 1단계 | 참여 점수, 자신감 추이, 성찰 내용 |
| student_notes (자율학습) | 1단계 | 자기주도 학습 빈도 |
| student_notes (실습일지) | 2단계 | 현장 실적 (상담/견적/수주), 성장 기록 |
| student_memos | 1~2단계 | 교육자 정성 관찰 기록 |
| student_questions | 1단계 | 질문 빈도, 적극성 |
| coaching_reports | 1단계 | 만성 오답(chronic), 성장 태그(overcome) |
| weekly_evaluations | 2단계 | R&P 강점/개선 태그, 영역 커버리지 |
| benchmarks | 2단계 | 벤치마킹 작성 여부, 실행력 |
| final_evaluations | 2단계 | 종합 평가 (5점 척도 × 5개 항목) |
| 수주 데이터 (수동) | 3단계 | 상담건수, 수주건수, 수주금액, 전환율 |
