# 프로젝트 로그 — 일룸 입문교육 대시보드

---

## 2026-03-31

### ✅ 오늘 한 것:

#### 1. 교육일지 하루 1개 제한 해제
- API(`/api/notes` POST)에서 하루 1개 중복 체크 로직 제거
- 프론트엔드에서 "오늘 일지 수정" 버튼 제거 → 항상 "새 교육일지" 버튼 표시
- 학생들이 하루에 자유롭게 여러 개 교육일지를 작성할 수 있게 됨

#### 2. 📚 자율학습 기능 추가
- **자율학습 토글 버튼**: 작성 폼 우상단에 `📚 자율학습` 버튼 추가
  - ON: 보라색 테마, STEP 1(학습 내용)만 표시, 자신감 선택 숨김
  - OFF: 기존 교육일지 모드 (STEP 1/2/3 + 자신감)
  - 안내 문구: "따로 공부한 내용은 자율학습으로!" 표시
- **자율학습 카드 UI 차별화** (학생 페이지 + 관리자 대시보드):
  - 보라색 테두리 + 보라색 배경
  - `📚 자율학습` 배지 표시
  - 참여점수, 자신감, STEP 아이콘 숨김
  - 태그도 보라색 톤으로 표시
- **제목 자동 생성**: 자율학습 모드 → `YYYY-MM-DD 이름 / 자율학습`

#### 3. 완료 체크 자동화
- 완료 체크 버튼 제거 → 내용이 있으면 자동으로 `✓ 작성됨` 배지 표시
- STEP 헤더: 내용 있으면 초록색 배경 자동 적용
- 참여점수: STEP 1/2/3 내용 유무로 자동 계산 (노션 체크박스 기준 아님)
- 카드 STEP 아이콘(📝💡🎯): 내용 기반으로 표시, 크기 11px → 15px

#### 4. 빠진 교육일지 전체 임포트
- 기존 임포트 스크립트가 "교육일지 기본 템플릿" 파일을 건너뛰던 문제 해결
- `계정:` 속성으로 교육생 매칭하는 새 스크립트(`reimport-all-notes.js`) 작성
- **19건 새로 추가** (채형우, 손희정, 최지혜, 김창석, 나원빈 등)
- 채형우 3/28 자율학습 노트 수동 임포트 (목재 소재·마감재·하드웨어 정리)

#### 5. 주말 노트 자율학습 자동 분류
- 토/일요일에 작성된 기존 노트에 `자율학습` 태그 자동 부여
- 곽현서 3/28, 3/29 업데이트 완료

#### 6. 관리자 대시보드 시간대 수정
- `created_at.slice(0, 10)` → `toKSTDate()` 함수로 변환
- UTC → 한국시간(KST) 기준 날짜 필터링
- 3/30 오전 내용이 3/29로 분류되던 버그 수정

#### 7. 최시원 날짜 불일치 수정
- 전체 마크다운 파일 대상 파일명 날짜 vs 실제 작성일 점검
- 최시원 2건 불일치 발견 및 수정:
  - 3/26: 스터디, 서재(다나) → 뉴트, 멘디 내용으로 교체 (실제 3/26 작성)
  - 3/27: 스터디, 서재(다나) 내용 새로 추가 (실제 3/27 작성)
- 나머지 1건(3/30 리빙)은 빈 노트라 영향 없음
- **다른 교육생은 날짜 불일치 없음 확인 완료**

- 완료 체크는 수동 아닌 내용 유무 자동 판정으로 변경
