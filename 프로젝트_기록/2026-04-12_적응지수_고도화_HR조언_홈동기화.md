# 2026-04-12 — 입문교육 적응 지수 고도화 + HR 조언 자동 생성 + 홈 대시보드 동기화

## ✅ 오늘 한 것:

### 1. 입문교육 적응 지수 — 5개 → 9개 항목으로 확장 후 8개로 정리

기존 5개 항목(시험/하위분야/출석/교육일지/자신감)만으로는 "조용히 무너지는" 학생을 못 잡아서 가중치를 재설계.

**최종 가중치:**

| 항목 | 가중치 | 무엇을 보나요 |
|------|-------|-------------|
| 시험 평균 | 30% | 전체 차시 시험 점수의 평균 |
| 하위 분야 | 15% | 60점 미만인 제품 카테고리 개수 (약점 카테고리 이름까지 노출) |
| 출석률 | 13% | 출석/지각/결석 (지각은 0.5로 계산) |
| 교육일지 참여 | 15% | 일지 제출과 STEP 작성 충실도 |
| 성장 기울기 | 10% | 최근 3차시와 초반 3차시 점수 비교 |
| 자신감 추이 | 8% | 일지 자신감 단계의 변화 (상승/유지/하락) |
| 만성 오답 | 5% | 계속 반복해서 틀리는 문항이 얼마나 되는지 |
| 메모 톤 | 4% | 교육자 칭찬/주의 메모의 비율 |

- 실습일지 가중치(3%)는 교육일지 참여와 같은 개념이라 제거 → 교육일지 참여 12% → 15%로 흡수
- 자신감 미입력 시 8% 가중치가 시험(+4%)과 교육일지 참여(+4%)로 자동 분산

### 2. 🆕 부정 신호 델타 감점 (최대 −8점)

점수가 괜찮아도 "조용히 무너지는" 학생을 놓치지 않기 위한 장치.

- 최근 시험 점수가 초반보다 **5점 넘게 떨어짐** → −5점
- 일지 자신감이 **계속 낮은 상태로 유지** → −3점

총점에서 직접 차감. 적응 상세 카드에 빨간 박스로 감점 사유 표시.

### 3. 🆕 HR 조언 자동 생성 — 8가지 유형

주의가 필요한 교육생 카드마다 맞춤 HR 조언이 자동으로 붙어요. 트리거된 체크 조합을 보고 유형을 판정.

| 유형 | 판정 조건 |
|------|---------|
| 🚨 복합 위기형 | 주의 신호 3개 이상 동시 해당 |
| 📘 지식 부족형 | 시험 낮음 + 하위 분야 많음 |
| 🎯 약점 편중형 | 시험은 양호한데 하위 분야만 많음 |
| 💔 심리 위축형 | 자신감이 계속 낮은 상태 |
| 🌊 하락 징후형 | 성적이나 자신감이 최근 하락 중 |
| 🚪 근태/동기 이슈형 | 출석률만 낮은 경우 |
| 🔍 행동 관찰형 | 주의 메모만 쌓인 경우 |
| 💎 부분 주의형 | 적응 지수 '상' 그룹인데 한두 가지 주의 |

각 유형마다 "어려움 서술 + 권장 액션 2~3개"가 자동 생성. 권장 액션에는 실제 약점 카테고리 이름, 하락폭, 자신감 수치 등이 꽂혀서 구체적으로 나옴.

**예) 복합 위기형 학생의 권장 액션 (데이터 기반 동적):**
1. 이번 주 내 긴급 1:1 면담 — "시험 평균 54.6점" 부터 직접 확인
2. 학생방/서재/다이닝 외 5개 집중 1:1 튜터링 (주 2회) — 암기 말고 개념 중심으로 다시
3. 최근 성적 -9.7점 원인 구체적으로 물어보기 — 학습/관계/개인사 모두 체크
4. 자신감 회복 면담 — 잘하는 점 3가지 구체적으로 짚어주고 작은 과제로 성공 경험 쌓기
5. HR팀에 상황 공유 + 주 2회 체크인 일정 확정 (중도 이탈 방지)

### 4. 🆕 "최근 하락 신호" 체크 + 🌊 하락 징후형 유형

점수가 '상' 그룹이어도 최근 성적/자신감 하락 중이면 잡아내는 새로운 체크 항목. 상 그룹 학생이 이 체크에 걸리면 파란 "부분 주의" 대신 주황 "🌊 하락 경고"로 강조 표시.

이를 통해 "잘하던 애가 번아웃으로 무너지는" 사각지대를 잡을 수 있음.

### 5. 계산 기준 / 판정 기준 드롭다운 UI

기존엔 카드 설명 문구가 너무 길어서 보기 나빴음. 오른쪽 상단 **ℹ️ 계산 기준** / **ℹ️ 판정 기준** 드롭다운으로 이동하고 표 형태로 정리:

- **적응 지수 드롭다운**: 8개 항목 가중치 표 + 부정 신호 감점 설명
- **주의 교육생 드롭다운**: 2단 구성 — ① 6가지 체크 신호 표 + ② 8가지 유형 판정 표

비개발자도 "왜 이 학생이 이 유형으로 분류됐는지" 바로 이해 가능.

### 6. 주의 교육생 카드 UI 정리

- HR 조언 박스의 점선 테두리 제거 → 배경색으로만 구분
- 바깥 카드의 빨간 외곽선 제거 → 연한 회색 테두리(적응 지수 카드와 통일)
- 주의 1개 해당 카드는 `--bg-elevated`로 바깥 카드와 시각 구분
- 복합 위기형 액션에서 이모지 prefix 제거 (🚨/📚/📉 등) → 번호 + 텍스트만
- "🎯 권장 접근" 중복 라벨 제거
- 카테고리 나열 스타일 통일: `학습/관계/개인사` (공백 없는 깔끔 버전)

### 7. 🆕 홈 대시보드 주의 교육생 동기화

기존 홈의 "주의 교육생"은 시험 점수만 기준이라 종합 분석 탭과 달랐음. 둘 다 **같은 6개 체크 + HR 조언 로직** 사용하도록 통일:

- 홈 카드 컴팩트 표시: 이름 + 적응 점수 + 체크 개수 + 유형 뱃지
- 🌊 하락 징후형까지 홈에서 바로 보임
- 정렬: 체크 개수 내림차순 → 적응 점수 오름차순 (가장 위험한 학생이 맨 위)
- 클릭 → `/dashboard/students?tab=analysis`로 이동 (URL 쿼리로 탭 전환)
- 카드 헤더에 "종합 분석 →" 링크 추가

홈 왼쪽 컬럼 순서도 재정렬: **주의 교육생 → 차시별 평균 → 평균 추이** (중요도 순)

## 🤔 결정한 것:

- 실습일지는 교육일지 참여와 동일 개념 → 가중치 합병
- 자신감 미입력 시 해당 가중치를 다른 항목으로 자동 분산 (중립 50 처리 폐기)
- 부정 신호 델타 감점은 총점에 직접 반영, 상세 카드에 별도 박스로 표시 (숨김 X)
- 상 그룹 학생이라도 하락 신호 있으면 🌊 하락 징후형으로 별도 케어 유도
- HR 조언 액션에서 이모지 prefix는 제거 (번호 붙이기만으로 충분)
- 홈과 종합 분석의 판정 로직은 100% 일치 → 혼란 방지
