# 2026-03-31 — 교육일지 전면 개편

## ✅ 오늘 한 것

### 1. 교육일지 표시 버그 수정
- 학생 페이지에서 노션 임포트(steps) 교육일지가 JSON 원본 대신 STEP 1/2/3 구조로 정상 표시
- content_type 전달 + StepsRenderer 추가

### 2. 교육생 STEP별 직접 작성 기능
- 기존 블록 에디터 → STEP 1/2/3 구조 에디터로 전면 교체
- 각 STEP마다 설명 문구 + 텍스트 영역 + "완료 체크" 버튼
- 서식 도구 바 추가 (● 목록 / B 굵게 / # 제목 / 1. 번호)
- 수정 기능 + 하루 1개 제한 + "오늘 일지 수정" 버튼

### 3. 제목 자동 생성 + 오늘의 다짐
- title: `YYYY-MM-DD 이름 / 교육일지` 자동 생성
- "오늘의 다짐" 선택 입력 (안 쓰면 자동 제목)

### 4. 참여점수 자동 계산
- STEP 내용 유무 기반 (0~3), completed 플래그 무관
- 3/3이면 ⭐ 우수학습 자동 부여
- 0점이면 미표시

### 5. 마크다운 렌더링
- 불릿 리스트 (`- 항목`), 표 (`| col |`), 제목 (`###`), 번호 리스트 (`1.`)
- **볼드**, *기울임* 인라인 서식
- 멀티라인 테이블 셀 병합
- 검색 하이라이트 형광색

### 6. 노션 임포트 파서 전면 수정
- aside 밖 STEP 헤더만 매칭 (안내문 aside 안의 [STEP] 참조 무시)
- aside 안에 `---` 있어도 섹션 안 잘림 (다음 STEP 헤더까지)
- 안내문 텍스트 필터링 (속성체크/반가워요/자유로운 학습 기록 등)
- fallback: aside 밖 텍스트도 추출 (안내문 제외)
- 75건 + 3/30 업데이트 재임포트 완료, 안내문 포함 0건 확인

### 7. 관리자 교육일지 UI 개선
- ⭐ 우수학습 배지 추가
- 배지 고정 슬롯 정렬: 태그 → 이해도(이모지) → 점수(⭐3/3)
- 모바일 반응형 (640px 이하: 14px, 태그 숨김)
- 필터 추가: 전체 / ⭐ 우수 / 미완료

### 8. 기타
- one_word 표시 제거 (우수학습 배지로 대체)
- 자동 제목 감지 제거 (### 마크다운 헤딩만 제목)
- 줄 간격 조정 (lineHeight 1.55, gap 축소)

## 🤔 결정한 것
- 노션 임포트 데이터는 이대로 두고, 앞으로는 앱에서 직접 작성
- 참여점수는 내용 유무 기반 (completed 플래그 무시)
- 제목은 자동 생성, 다짐은 선택 입력
- 마크다운 자동 제목 감지 제거 (### 헤딩만 제목 처리)
