# 2026-04-18 — 시험 분석 탭 전면 개편 (4대 신기능 + UX 재설계)

## ✅ 오늘 한 것

### 1. 시험 분석 탭 4가지 신기능 추가

#### 🎯 오답 선지 분포
- 각 문항마다 **오답자들이 어떤 답을 골랐는지** 분포를 막대그래프로 표시
- 상위 5개 답 · 색상 강조 (1위만 orange, 나머지 text-muted)
- 한 답에 60% 이상 쏠리면 **"집중 오답"** 뱃지 — 개념 오해 신호
- 답 번호 추출 로직: `N)` 패턴만 매칭 (본문 안 숫자 제외)
- 긴 선지 텍스트는 바 아래에 분리 표시 (바 너비 찌그러짐 방지)

#### 🚩 문항 퀄리티 플래그
- 정답률 10% 미만 → **"문항 재검토"** 빨간 뱃지 (문제 자체 모호 가능성)
- 정답률 100% → **"너무 쉬움"** 회색 뱃지 (변별력 부족)

#### 🏁 기수간 비교
- 전체 평균 3칸 (현재 / 이전 / 차이) — 교육일지 compact 스타일
- 차시별 평균 비교 테이블 (전체)
- 제품군별 정답률 비교 그리드 (한 줄에 `75% vs 40% +35%p` 나열, diff 강조)
- 10%p 이상 차이 = 빨간 테두리 (이번 기수가 유독 약한 영역)
- 드롭다운으로 비교 대상 기수 변경

#### 🤖 AI 원인 분석 (Gemini 2.5 Flash)
- 각 문항 카드 하단 버튼으로 호출
- 4개 섹션 구조 (왜 틀렸나 / 문항 점검 / 보강 제안 / 이렇게 출제했다면)
- 마지막 섹션은 **문항 설계에 문제가 있을 때만** AI가 대안 문항 예시 제시 (문제·보기·정답·개선 포인트)
- `thinkingBudget: 0` 설정으로 thinking 비활성화 (응답 잘림 방지)
- `maxOutputTokens: 2048`로 증액
- 섹션 간 구분선 (`borderTop: 1px solid --border-light`)

### 2. 분석 탭 섹션 순서 재설계 — 교육자 사고 흐름 따라가기

**이전:** 기수간 비교(맨 위) → 추이+성장곡선 → 히트맵
**현재:** 스냅샷 → 추이+성장곡선 → 주의 문항 → 히트맵 → 기수간 비교

교육자가 페이지 열자마자:
1. **📍 지금 상황** (최근 평균 / 누적 평균 / 응시 현황 / 주의 교육생)
2. **📉 추이 + 성장 곡선** (시간 축)
3. **🚨 주의 문항 TOP 7** (액션 아이템)
4. **🎯 카테고리별 약점 맵** (영역 진단)
5. **🏁 기수간 비교** (역사적 맥락)

### 3. 차트 바 차트로 전환

- LineChart → BarChart (다른 페이지와 통일)
- 차시별 전체 평균 추이: 단일 시리즈 바, 전일 대비 3점↓ 하락 시 red 셀
- 교육생별 성장 곡선: 그룹 바 차트, **전체 평균(회색 #d1d5db) 왼쪽 + 개인 점수(파랑/빨강) 오른쪽** (교육생 상세 페이지 패턴과 동일)
- `maxBarSize={20}` 통일 — 바 두께 일관
- **막대 클릭 → 상세 모달 바로 열림** (날짜 버튼 제거, UX 단순화)

### 4. DESIGN_SYSTEM 전면 적용

#### 이모지 제거
- 섹션 타이틀에서 📍 🚨 🎯 🏁 📉 👥 전부 제거
- 뱃지에서 🚩 ⚪ 🎯 🤖 제거 — 색상 + fontWeight 위계로 대체
- AI 응답 내 이모지는 유지 (DESIGN_SYSTEM §17 AI 파싱 패턴 예외)

#### 폰트 위계 정리
- 섹션 타이틀: 17px/700 → 20px/700 -0.015em (h3 스펙)
- 숫자: 24-32px / 800 -0.02em
- 스탯 라벨: 13px / 500 text-tertiary
- 오답 선지 분포 내 위계: 1위 700 / 나머지 500 / 2위 이하 text 색상 연하게

#### 모달 스펙 (§17-D)
- 배경: 솔리드 `--bg-surface` (반투명 금지)
- 닫기 버튼: 36×36 원형 + `×` (fontSize 20, lineHeight 36)
- 헤더 paddingRight: 44 (닫기 버튼 자리)

#### 컴팩트 요약 패턴 (교육일지 스타일)
- 이번 기수 스냅샷: borderRight 구분선으로 4칸, border-top/bottom 제거
- 학생별 모달 요약: 3칸 + `--bg-main` 배경
- 하단 구분선: 숫자 행과 주의 교육생 리스트 사이만 유지
