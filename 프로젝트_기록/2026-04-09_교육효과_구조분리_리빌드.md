## ✅ 오늘 한 것:

### 1. "교육생 분석" vs "교육 효과 분석" 구분 정립
- 기존 `/dashboard/analytics`에 교육생 개인 분석과 교육 프로그램 효과 분석이 혼재
- 교육생 분석 = "이 교육생이 잘하고 있나?" (개인 진단 → 면담/개입 도구)
- 교육 효과 분석 = "이 교육 프로그램 자체가 효과 있나?" (프로그램 개선 → 2기 설계 근거)
- **수지님 판단**: 교육효과분석은 프로그램 관점이 맞다. 교육생 분석은 기존 페이지로 이동

### 2. 테스트 페이지 (`/dashboard/tests`) — "시험 분석" 탭 추가
- "차시별 성적" | "시험 분석" 탭 토글 추가
- 시험 분석 탭에 이동된 섹션:
  - 차시별 전체 평균 추이 (라인 차트 + 날짜 버튼, 3점↓ 하락일 빨간 강조)
  - 교육생별 성장 곡선 (개인 vs 전체 평균 비교, 드롭다운 선택)
  - 카테고리별 약점 맵 (대분류 × 소분류 히트맵, 셀 클릭 → 문항 상세)
- 모달 3종 그대로 이동: 날짜별 상세, 학생별 날짜 분석("나만 틀린 문항"), 히트맵 문항 상세
- page.tsx: test_responses 전체 페이지네이션 + questions + attendance + notes 추가 fetch
- 기존 차시별 성적/문제은행/채점 기능 100% 유지

### 3. 교육생 페이지 (`/dashboard/students`) — "종합 분석" 탭 추가
- "명단" | "종합 분석" 탭 토글 추가

  - 입문교육 적응 지수 (상/중/하 카드 그리드, 클릭 → 5항목 상세 근거)
  - 주의가 필요한 교육생 (5개 위험 항목 체크, 적응 "상"이면 "부분 주의" 파란 카드)
  - 태도/참여 현황 (출석률/제출률/참여점수 KPI 카드 + 자신감 추이 스택바)

- 기존 명단 테이블/필터/퇴사자 기능 100% 유지

### 4. 교육효과 페이지 (`/dashboard/analytics`) — 프로그램 관점 리빌드
- 가상 수주 데이터 생성 (14명 × 4주, 시드 기반 결정적 랜덤)
- "🧪 가상 수주 데이터로 미리보기" 배지로 가상 데이터 명시
- **교차 분석 4종**:
  1. 시험 성적 그룹(상/중/하) vs 수주 전환율/금액 → "시험이 실전을 예측하나?"
  2. 카테고리별 정답률 vs 수주 분야 금액 → "잘 아는 분야를 잘 팔까?"
  3. 적응 지수 vs 수주 전환율 (교육생별 카드) → "적응 지수로 위험 교육생 조기 발견 가능?"
  4. 태도(출석률/일지 제출률) vs 수주 전환율 → "성실한 교육생이 수주도 잘하나?"
- 주간 수주 추이: 1인 평균 금액 차트 + 주간 전환율 카드
- 2기 교육 설계 핵심 질문 6개 카드 (실제 수주 데이터 들어오면 답할 수 있는 질문)
- 인사이트 자동 생성: 상위/하위 그룹 간 차이 분석 텍스트
