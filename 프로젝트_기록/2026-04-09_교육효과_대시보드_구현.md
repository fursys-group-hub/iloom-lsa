# 2026-04-09 — 교육 효과 대시보드 구현

## ✅ 오늘 한 것:

### 1. 교육 효과 대시보드 페이지 신규 생성 (`/dashboard/analytics`)
- 사이드바에 📊 교육효과 메뉴 추가
- Server Component(page.tsx) + Client Component(AnalyticsClient.tsx) 구조
- Supabase 10개 테이블 병렬 fetch (test_responses는 6629건 전체 페이지네이션)

### 2. 교육생 종합 현황 — 입문교육 적응 지수
- 시험평균(40%) + 하위분야(20%) + 출석률(15%) + 교육일지참여(15%) + 자신감추이(10%)
- 카드 클릭 시 항목별 근거 상세 펼침 (게이지바 + 설명 텍스트)
- "하" 그룹 교육생은 빨간 배경으로 즉시 식별

### 3. 위험 교육생 알림 — 카드 형태
- 5개 항목: 시험평균 60점-, 하위분야 4개+, 출석률 80%-, 자신감 3회연속 낮음, 주의메모 2건+
- 자신감 미입력 → "미입력" 표시 + 위험 판정 제외
- 질문/자율학습 항목 제거 (정확도 낮음)
- 적응 지수 "상"인 교육생은 파란 카드 + "부분 주의" 뱃지
- 양호 항목은 줄바꿈으로 가독성 향상, 영어(understood) → 한국어(보통) 변환

### 4. 시험 성적 추이 + 하락 분석
- 차시별 전체 평균 추이: 최근 평균 + 변화 수치 표시
- 교육생별 성장 곡선: 개인(파란선) + 전체 평균(회색선) 동시 표시
- 날짜 버튼: 하락 큰 날 빨간 표시, 클릭 시 상세 분석 모달
- **전체 하락 분석 모달**: 시험 분야 + 문항별 정답/오답 수 + 문항 텍스트 + 교육생별 점수 카드
- **교육생 개인 분석 모달**: "다른 애들은 맞췄는데 이 학생만 틀린 문항" + 약점 패턴(대분류/소분류) + 교육 제안

### 5. 카테고리별 약점 맵 (히트맵)
- CSS Grid: 가로=대분류(제품군), 세로=소분류(지식유형)
- 정답률별 색상: 80%+ 초록, 60~79% 주황, 60%- 빨강
- 축 설명 + 범례 추가
- 셀 클릭 시 해당 문항 목록 팝업 (차시, 문항번호, 정답/오답 수, 오답률)

### 6. 태도/참여 현황
- 통계 카드 3개: 평균 출석률, 교육일지 제출률, 평균 참여 점수
- 자신감 추이 차트 (일별 높음/보통/낮음 비율)

### 7. 수주 실적 placeholder
- 수주 실적 + 수주 분야별 분석 → "4/23 이후 활성화" 잠금 카드

### 8. 버그 수정
- Supabase 기본 limit 1000건 → test_responses 6629건 전체 페이지네이션으로 수정
- 부동소수점 오류 (-13.200000000000003 → -13.2) 수정
- understood → 보통 한국어 변환 (자신감 추이 전체)
