# 2026-04-14 — 홈 대시보드 리디자인 + UI/UX 감사 리포트 시행

## ✅ 오늘 한 것:

### 1. 홈 대시보드 리디자인
- **출결 요약 카드**: 결석/지각자 이름을 숫자 카드 바로 아래에 빨간 뱃지로 표시
- **차시별 평균**: 가로 막대 → Recharts 세로 막대 차트로 변경 (색상별 점수 구분)
- **교육일지 카드**: 어제/오늘 2컬럼 레이아웃 (회색 박스로 구분)
  - 어제: 제출 12 / 미제출 2 + 미제출자 이름 뱃지
  - 오늘: 일정 타입 + 제출 현황
- **"최근 공지" 카드 제거** (내가 쓴 걸 내가 볼 필요 없음)
- **"최근 교육생 답글" → 질문관리에 통합** ("답글 N건" 뱃지로 표시)
- **"평균 추이 차트" 제거** (차시별 평균 바 차트와 중복)
- **2컬럼 재배치**: 왼쪽(차시별 평균 + 교육일지 + 질문관리) / 오른쪽(주의 교육생 + 실습일지 실적)
- **안쪽 박스 배경색**: `bg-elevated` → `bg-main`으로 변경 (뱃지 가독성 개선)

### 2. 교육생 상세 페이지 강화
- **탭 추가**: 요약 | 출결 | 일지 | 질문
- 출결 탭: 출결 이력 테이블 (날짜, 상태 뱃지, 비고)
- 일지 탭: API로 교육일지 + 실습일지 로드, 펼치기/접기 아코디언
- 질문 탭: 질문 스레드 + 답글 이력 (관리자 답글은 파란 배경)

### 3. 질문 페이지 학생 컨텍스트
- 대화 헤더에 "개별분석 보기 →" 링크 추가

### 4. 코멘트 말풍선 확인
- 6개 파일 전수 확인 → 모두 올바름 (수정 불필요)

### 5. CSS/반응형 정리
- 레거시 CSS 클래스 제거 (`date-buttons-desktop`, `date-select-mobile`)
- 주의 교육생 그리드 gap 6→8px

### 6. 스케줄 날짜 KST 대응
- 스케줄 조회용 `kstToday`/`kstYesterday` 별도 변수 추가
- 기존 `today`/`yesterday`(UTC 기반)는 출결/노트 필터링에 그대로 유지
- 스케줄 키가 KST 날짜로 저장되어 있어서 스케줄 조회에만 KST 사용
