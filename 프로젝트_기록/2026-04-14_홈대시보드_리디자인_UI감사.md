# 2026-04-14 — 홈 대시보드 리디자인 + UI/UX 감사 리포트 시행

## ✅ 오늘 한 것:

### 1. 홈 대시보드 리디자인 (대규모)
- **교육 일정 달력** 신규 추가
  - 파란 패널(날짜+일정+할일) + 흰색 달력 그리드 2분할 카드
  - 날짜 클릭 → 해당 날짜 일정(정규교육/매장실습/테스트/공지) 표시
  - 할 일 메모: 체크박스(줄긋기), 더블클릭 수정, × 삭제
  - 이벤트 점: 파랑(정규), 주황(실습), 초록(테스트), 빨강(공지), 보라(메모)
  - Supabase `calendar_memos` 테이블 연동 (어디서든 접근 가능)
- **출결 요약**: 오른쪽 컬럼 상단에 배치, 결석/지각자 이름 뱃지
- **차시별 평균**: Recharts 세로 막대 차트 (점수별 색상 구분)
- **교육일지 카드**: 어제/오늘 2컬럼 회색 박스 레이아웃
  - 어제: 제출 12 / 미제출 2 + 미제출자 이름 뱃지
  - 오늘: 일정 타입 + 제출 현황
  - 스케줄 조회 KST 전용 변수 (`kstToday`/`kstYesterday`) 분리
- **실습일지 실적**: Sales Funnel 스타일 (상담→견적→수주 숫자 + 영역 차트)
- **카드 접기/펼치기**: 차시별 평균, 교육일지, 질문관리, 주의 교육생, 실습일지 — 5개 카드 ▼ 토글
- **제거**: 최근 공지 카드, 최근 교육생 답글 카드, 평균 추이 차트 (차시별 평균과 중복)
- **통합**: 교육생 답글 → 질문관리 카드에 "답글 N건" 뱃지로 통합
- **안쪽 박스 배경색**: `bg-elevated` → `bg-main`으로 변경 (뱃지 가독성 개선)

### 2. 교육생 상세 페이지 강화
- **탭 추가**: 요약 | 출결 | 일지 | 질문
- 출결 탭: 출결 이력 테이블 (날짜, 상태 뱃지, 비고)
- 일지 탭: API로 교육일지 + 실습일지 로드, 펼치기/접기 아코디언
- 질문 탭: 질문 스레드 + 답글 이력 (관리자 답글은 파란 배경)

## 🤔 결정한 것:
- 출결 도넛 차트 시도 → 공간 낭비로 판단하여 원래 숫자 카드로 복원
- 교육일지 "어제/오늘" 데이터 매핑: 스케줄은 KST 전용 변수, 노트/출결은 기존 UTC 유지
- 테스트+교육효과 관계 정리: B안(교육 인사이트 리브랜딩) 추천했으나 아직 미적용
- 왼쪽/오른쪽 컬럼 비율: 반반(1:1) 확정
- 달력 디자인: 여러 참고 이미지 반영 → 파란 패널 + 흰색 달력 2분할 확정
- 할일 메모: localStorage → Supabase 전환 (다른 기기에서도 접근 가능)
