# 2026-04-02 — Phase 3.5: 실습일지 + 질문관리 + UI 전면 정비

## ✅ 오늘 한 것 (오전 세션):

### 1. 서버 날짜 계산 KST 통일
- Railway 서버(UTC)에서 한국 시간과 9시간 차이로 교육일지 중복 판정 오류 발생
- `lib/date.ts` 유틸 함수 추가 (`getKSTToday()`, `getKSTDayRange()`)
- 4개 API 일괄 수정: notes, sync, students, export-tests

### 2. 교육일지 이미지 첨부 기능
- `/api/upload`: Supabase Storage(`note-images` 버킷) 이미지 업로드 API (5MB 제한)
- 교육생 작성 UI: 각 STEP 하단에 📷 사진 첨부 버튼 + 썸네일 미리보기 + 삭제
- 관리자/교육생 조회: 이미지 그리드(120px) + 클릭 시 전체화면 라이트박스(이전/다음)
- 이미지 URL은 기존 JSON content에 `step1_images`, `step2_images`, `step3_images` 배열로 저장 (DB 스키마 변경 없음)

### 3. 노션 기존 이미지 업로드
- `scripts/upload-note-images.js` 스크립트 작성
- 손창환 6장, 안덕찬 1장, 곽현서 1장 = 8장 Supabase Storage 업로드 완료
- 나원빈 63장은 제품별 노션 페이지 사진(교육일지 아님)이라 제외
- 손창환 3/27 image 55.png (8.7MB) 1장은 5MB 제한 초과로 스킵

### 4. 실습일지 기능 추가
- `/my/practice` (학생), `/dashboard/practice` (관리자) 신규 페이지
- 4개 텍스트 섹션: 기억에 남는 고객 / 선배 비법 / 칭찬할 점 / 보완할 점
- 4개 숫자 필드: 상담건수, 견적건수, 수주건수, 수주금액 (콤마 자동포맷)
- 상담/수주 내역 선택 필드 추가
- API: 실습일지 태그 지원, 교육일지/실습일지 중복체크 분리
- 관리자: 날짜별 필터, 전체 실적 합계, 제출현황, 코멘트 지원
- 교육일지 페이지에서 실습일지 노트 제외 필터링

### 5. 홈 대시보드 리뉴얼
- D-day 뱃지 추가 (입문교육 몇일차, 종료까지 D-n)
- 2컬럼 레이아웃: 차시별 평균 + 추이 차트 + 주의 교육생 / 교육일지 제출현황 + 실습일지 실적 + 최근 공지 + 교육생 답글
- 교육일지 제출현황: "제출 5 / 미제출 9" 한줄 + 미제출자 뱃지
- 실습일지 실적: 상담/견적/수주/금액 미니카드

### 6. 교육일지 UI 개선
- 요약 카드 그리드화 (제출현황 + 참여도 + 추가 설명 필요 교육생)

### 7. 퇴사자 처리 통일

### 10. 질문하기 기능 추가 (학생↔관리자)
- **학생 페이지** `/my/ask`: 왼쪽 질문 목록 + 오른쪽 채팅 영역

  - 상태 필터(답변대기/답변완료), 자기 답변 수정/삭제
- 상태 흐름: 대기 🟠 → 관리자 답변 → 답변완료 🟢 (2단계)
- hover 시 수정/삭제 팝업 (양쪽 다 자기 메시지만)
- DB: `student_questions` + `question_replies` 테이블
