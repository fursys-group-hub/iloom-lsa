# 2026-04-16 — 교육설문 관리자 페이지 + 재촉 알림 시스템 + 공통 배너 컴포넌트

## ✅ 오늘 한 것:

### 1. 교육설문 관리자 페이지 신설 (`/dashboard/surveys`)
- 기존에는 교육생 설문 응답을 관리자가 볼 방법이 없었음 → 새 페이지 개설
- **2개 탭 구조**: 안성공장 투어 / 자기효능감 설문
- **1줄 요약 카드** (심화교육 스타일 적용): 제출수/평균/NPS 등 핵심 지표
- **레이더 차트 + 세로 바 차트**:
  - 레이더: 사전/사후 값 겹쳐서 전체 패턴 비교
  - 바 차트: 항목별 사전 vs 사후 나란히 (recharts `maxBarSize={28}`로 너비 제한)
  - 사후 바 위에 `+X.X` 델타 라벨 표시
- **자연어 인사이트 헤드라인** (분석 페이지 스타일):
  - 데이터 기반 자동 생성: "공장 규모가 가장 자랐지만, 철제 공정은 상대적으로 덜 자랐어요"
  - 긍정 + 아쉬움을 한 문장으로 엮음 (핵심 1줄만)
  - 타입별 문구 차별화 (efficacy/edu-satisfaction/ansan-knowledge/ansan-satisfaction)
- **제출 현황 컴팩트 카드 그리드**:
  - 2행 구조: 상단(아바타 + 이름 + 전체 상태 뱃지) / 하단(phase별 pill)
  - 외곽선 제거(흰 카드 안이라) + borderBottom으로 행 분리
  - 클릭 시 상세 모달(응답 내용 + 질문 원문까지 표시)

### 2. 미제출자 재촉 알림 시스템
- **새 테이블**: `survey_reminders` (student_id / survey_type / phase / dismissed_at)
- **API**: `/api/survey-reminders` GET/POST/PATCH (upsert, dismissed 처리)
- **관리자 측 일괄 전송**:
  - 제출 현황 헤더 우측에 phase별 빨간 버튼: "📢 입문 미제출 N명 재촉" / "📢 심화 미제출 N명 재촉"
  - 입문/심화 구분해서 따로 발송 (두 phase 합쳐서 보내면 안 되므로 분리)
  - 확인 모달 후 일괄 POST
- **교육생 측 강조 배너**:
  - 홈 할 일 섹션, 교육일지/실습일지 미작성 알림 바로 밑에 배치
  - 빨간 테두리 1.5px + red-dim 배경 + shadow → 강조
  - "안성공장 투어 사후 설문을 제출해주세요" + "지금 쓰기 →" 빨간 버튼
  - 텍스트는 `whiteSpace: 'nowrap'`로 1줄 고정

### 3. 공통 `<Banner>` 컴포넌트 (`components/Banner.tsx`)
- 기존 매장실습일/휴무일 배너가 레이아웃이 달라서(휴무일은 세로 정렬) 통일 필요
- **API**: `icon`, `title`, `description`, `tone` (6색), `actionText`, `actionHref`, `onClose`
- **공통 규격**: 1줄 레이아웃(제목 왼쪽, 버튼 오른쪽) + 버튼 수직 중앙 정렬(`inline-flex` + `lineHeight: 1.2`)
- 휴무일 배너를 orange에서 purple로 변경 (자율학습 톤 = 보라)

### 4. 사이드바 그룹 재정리
- 그룹 라벨 띄어쓰기 제거: "일일 운영" → "일일운영", "성과 분석" → "성과분석"
- **일일운영**: 공지사항(맨 위) + 출결/테스트/교육일지/실습일지/질문관리 + AI 리포트
- **성과분석**: 교육생분석 → 교육설문 → 교육인사이트 (분석 흐름에 맞게 재정렬)
- **설정**: 기수 관리만 (공지사항은 일일운영으로 이동)
- 교육생 할일 페이지는 사이드바에서 제거 (관리자가 학생 개인 메모장 볼 필요 없다고 판단)

### 5. 심화교육 매트릭스 개선
- `/dashboard/overview` 테이블에 "벤치마킹" 열 추가
- 교육생별 작성 건수 표시 + 4건 이상이면 초록 뱃지

### 6. 기타 UI 정비
- 페이지 타이틀 통일: 신규 페이지도 `fontSize: 28, fontWeight: 700`로 맞춤 (기존 h1 clamp → h2 28px)
- 재촉 버튼: 처음엔 개별 카드당 1개 → 헤더 일괄 전송 버튼 1개로 재설계
- 교육생 홈 휴무일 배너 버튼 수직 정렬 버그 수정

## 🤔 결정한 것:

1. **교육설문은 '성과분석' 그룹** — 교육생이 설문을 "하는" 건 My 쪽이고, 관리자는 결과를 "분석"하는 거니까
2. **교육생 할일은 관리자가 안 본다** — 공식 제출물이 아닌 개인 메모장이므로 감시하는 느낌. 페이지 파일은 남겨두고 사이드바에서만 제거
3. **매니저 평가 → 교육생 열람은 안 만든다** — 수지님 본인이 원치 않음 (교육생이 자기 평가 보는 건 아닌 것 같다)
4. **재촉 알림은 phase별 분리 발송** — 입문 미제출자와 심화 미제출자 컨텍스트가 다르므로 한 버튼에 섞지 않음
5. **인사이트 헤드라인은 1개만** — 여러 개 나열하면 같은 항목 중복되거나 정보 과잉. 긍정 + 아쉬움을 한 문장으로 엮는 패턴 고정
6. **배너 컴포넌트 템플릿화** — 향후 공지 팝업 등에도 재사용
