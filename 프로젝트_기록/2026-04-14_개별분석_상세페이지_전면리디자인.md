# 2026-04-14 — 개별분석 + 상세페이지 전면 리디자인

## ✅ 오늘 한 것:

### 1. 개별분석 페이지 리디자인
- 명단 탭 제거, 종합분석만 유지
- 적응지수 카드 → **프로필 카드 3열** (사진+생년+학력+시험평균+적응지수)
- **HR 유형 뱃지** 카드에 직접 표시 (하락 징후형, 복합 위기형, 약점 편중형 등)
- **복합 위기형 카드 빨간 배경** 강조
- 주의가 필요한 교육생 섹션 → 카드 뱃지로 통합 (별도 섹션 제거)
- 카드 전체 클릭 → 상세 페이지 이동
- 카드 스타일: 외곽선 없이 `bg-main` 배경 + hover 효과

### 2. 상세페이지 2열 레이아웃
- **5개 탭 제거** → 홈 대시보드 스타일 2열 레이아웃
- 왼쪽: 적응지수 레이더 차트 + 항목별 표 + 출결 이력 
- 오른쪽: 차시별 점수 바 차트 + 카테고리별 학습 + 교육일지 + 실습일지 + 질문

### 3. 프로필 카드 통합 (프로필 + HR 조언)
- 왼쪽: 사진 + 이름 + 생년(나이) + 학력/경력 2열 (날짜|학교|전공 파싱)
- 오른쪽: HR 조언 (하락 징후형 등) — 어려움 + 권장 액션
- 모바일: 1열로 쌓임 (profile-grid 반응형)

### 4. 적응지수 산정 근거 리디자인
- 8개 바 차트 → **레이더(거미줄) 차트** + **항목별 표**
- 표: 항목 | 점수(색상) | 상세 — 초록(75+)/주황(60~74)/빨강(60 미만)
- 적응지수 기준: 상 75→**70**점으로 완화

### 5. 차시별 점수 추이
- 라인 차트 → **바 차트** (반 평균 회색 + 본인 파랑/빨강)
- 반 평균보다 높으면 파란색, 낮으면 빨간색

### 6. 출결 이력 개선
- 날짜/상태/비고 → **날짜/출근/퇴근/상태** (관리자 출결 화면과 동일)
- note에서 출근/퇴근 시간 자동 파싱
- 제목 옆에 **출석률 + 반 평균** 표시

### 7. 교육일지 현황
- 제목 옆에 **제출률 + 참여점수 + 반 평균** 표시
- 카드 외곽선 → 배경색(`bg-main`)으로 변경

### 8. 실습일지 전환율 (Sales Funnel)
- 상담→견적→수주 **퍼널 차트** (세로 구분선 + 전환율 크게)
- 상담(건수) 100% | 견적(건수) 71% | 수주(건수) 43% | 수주금액
- 각 단계별 전환율 색상: 초록(50%+)/주황(30%+)/빨강

### 9. 교육자 메모 리디자인
- 프로필과 2열 사이에 **독립 카드**로 배치
- 카테고리 버튼: 외곽선 → `bg-main` 배경, `radius-sm`
- 입력필드: 외곽선 → `bg-main` 배경
- 메모 타임라인: 회색 박스 → **한줄 표시** (뱃지 | 내용 | 날짜 | ×)

### 10. DB 확장 + 데이터 입력
- `students` 테이블에 `photo_url`, `birth_date`, `education`, `experience` 컬럼 추가
- 전체 교육생 14명 학력/생년월일 일괄 입력
- 곽현서/이예나 경력 상세 입력 (날짜|회사|직무 형식)
- 반 평균 비교용 `allAttendance`, `allNotes` 데이터 추가 전달

### 11. 일지 API 버그 수정
- `studentId` 필터가 `all=true`일 때 무시되던 문제 수정
