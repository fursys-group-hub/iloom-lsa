# 2026-04-09 — 자신감 5단계 필수화 + 질문 보관처리 + 스케줄 시스템

## ✅ 오늘 한 것:

### 1. 자신감 4단계 → 5단계 + 필수 입력으로 변경
- **기존 4단계**: 자신만만 / 이해완료 / 알쏭달쏭 / 도움요청
- **새 5단계**: 😎자신만만(5) / 😊자신있어요(4) / 😐보통이에요(3) / 🤔알쏭달쏭(2) / 😵도움요청(1)
- 선택 → **필수**로 변경: 미선택 시 저장 버튼 비활성화 + "자신감을 선택해주세요" 안내
- 미선택으로 저장 시도 시: 자신감 영역으로 **자동 스크롤 + 흔들림(shake) 애니메이션 + 빨간 테두리**
- 자신감 선택하면 에러 자동 해제
- **기존 데이터 완벽 호환**: 구 confident→very_confident(5점), 구 understood→confident(4점), 구 half→uncertain(2점)
- 분석 로직(analysis.ts), 대시보드 차트(StudentsClient.tsx), 교육일지 관리자 뷰 전부 5단계 반영

### 2. 질문관리 보관처리 기능
- `student_questions.status`에 `archived` 상태 추가
- **관리자**: 대화 헤더에 📦보관 / 보관 해제 버튼
- **관리자**: "전체" 탭에서 보관된 질문 숨김, "📦 보관됨" 탭에서만 표시
- **교육생**: 보관된 질문 선택 시 "📦 이 질문은 보관처리 되었습니다" 안내 + 입력 영역 차단
- 상태 뱃지: "보관처리 됨" (회색)

### 3. 교육일지 날짜 수정 (관리자용)
- 관리자 교육일지 페이지에서 노트 펼치면 "날짜 수정" 버튼 표시
- 클릭 → date picker → 확인 → `created_at` 즉시 수정
- 기존 "작성: 2026. 4. 8. 오후 12:00:00" → **날짜만 표시** (시각은 전부 정오로 고정되어 있어 무의미)
- API: `PATCH /api/notes`에 `created_at` 수정 지원 추가

### 4. 교육 스케줄 시스템 (A안)
- `batches` 테이블에 `schedule` JSONB 컬럼 추가
- 1기 스케줄 데이터 저장: 정규교육 19일 / 매장실습 4일 / 휴무 8일
- **스케줄 구조**: `{ "2026-03-23": "education", "2026-04-10": "off", "2026-04-12": "practice", ... }`
- **3가지 날짜 유형**:
  - `education` (정규교육) → 교육일지 + 자율학습 작성 가능
  - `practice` (매장실습) → 실습일지(별도 페이지) + 자율학습 작성 가능
  - `off` (휴무) → 자율학습만 가능
- **교육생 교육일지 자동 모드 전환**:
  - 날짜 드롭다운에 유형 표시: `4/12 (토) 🏪실습`, `4/10 (목) 🌙휴무`
  - 휴무일 선택 시 자동으로 자율학습 모드 전환
  - 매장실습일에는 "실습일지 작성하기 →" 링크 안내 배너
  - 폼 열 때 + 날짜 변경 시 자동 전환
- `lib/schedule.ts` 유틸 + `scripts/init-schedule.js` 초기화 스크립트
