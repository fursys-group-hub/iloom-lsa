- `입문교육 기록확인` → `입문교육 기록 확인` (띄어쓰기)

### 2. 뒤로가기 링크 숨김
- `StudentDetailClient`의 `← 교육생 목록` 링크를 `{!hideMemos && ...}`로 감싸 매장관리자 뷰에서 숨김
- 매장관리자는 어차피 탭 형태로 이동하므로 불필요

### 3. 민감정보 숨김 (`hideMemos={true}` 조건)
- 학력/경력 섹션 숨김
- 상/중/하 적응지수 뱃지 숨김
- HR 조언 (하락징후형 등) 숨김
- 적응지수 산정 근거 섹션 숨김

### 4. 차시별 점수 추이 바 클릭 → 오답 모달 팝업

#### 구현 흐름
- `barModalDate: string | null` 상태 추가
- `barModalData` useMemo: `responses.filter(r => r.test_date === barModalDate)`로 오답 추출
- 반드시 `sessionWrongs` useMemo **이후**에 선언해야 함 (초기화 순서 오류 방지)
- Recharts `BarChart onClick`은 불안정 → **`Bar` 컴포넌트에 직접 onClick** 달아야 작동
- `Bar onClick` 콜백은 `data as unknown as { rawDate: string }` 이중 캐스팅 필요 (BarRectangleItem 타입 제약)

#### 모달 UI
- 헤더: ✕ 아이콘 + `question_id · detail · earned_score/max_score점`
- 본문: 문제 텍스트
- 하단 2열: 학생 답안(빨간) | 정답(회색)

## 삽질한 것들

1. **Hook 초기화 순서 오류**: `barModalData` useMemo를 `barModalDate` state 바로 아래에 뒀다가 `sessionWrongs` 참조 불가 에러 → sessionWrongs 뒤로 이동
2. **`TestScore`에 session 없음**: `dailyAverages` 날짜와 `sessionWrongs` 차시를 인덱스로 연결하려 했으나 정렬 방향이 달라서 실패 → `test_responses`에 `test_date` 컬럼이 있다는 걸 확인하고 직접 날짜 필터로 전환
3. **`BarChart onClick` 미작동**: Recharts에서 BarChart 레벨 onClick은 실제 바 클릭 시 activePayload가 안 잡힘 → `Bar` 컴포넌트에 직접 onClick 추가로 해결
4. **TypeScript 타입 충돌**: `BarRectangleItem`에 `rawDate` 없음 → `as unknown as { rawDate: string }` 이중 캐스팅
