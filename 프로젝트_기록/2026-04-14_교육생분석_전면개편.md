# 2026-04-14 — 교육생 분석 페이지 전면 개편

## ✅ 오늘 한 것:

### 1. 교육생 비교 모드 (목록 페이지)
- "비교" 버튼 클릭 → 비교 모드 활성화, 카드에 체크박스 표시
- 2~3명 선택 시 하단에 비교 패널 렌더링
  - 레이더 차트 겹침 비교 (적응지수 8개 항목)
  - 항목별 비교 테이블 (시험평균, 출석률, 성장기울기, 교육일지 참여, 자신감 추이, 하위분야)
  - 가장 높은 값 강조 표시
- 각 학생별 색상 구분: blue / purple / orange
- "비교 해제" 클릭 시 원래 모드(카드 클릭 → 상세 이동)로 복원
- Recharts RadarChart import 추가

### 2. 교육 캘린더 + 메모 통합 (상세 페이지)
- 홈 대시보드 달력과 동일한 UI 구조 (왼쪽 파란 패널 + 오른쪽 7열 달력 그리드)
- 왼쪽 파란 패널:
  - 날짜 클릭 시 큰 숫자(56px) + 요일 표시
  - 해당 날짜의 출결(출근/퇴근 시간), 시험 점수, 교육일지 제출 여부, 메모 표시
  - 날짜 미선택 시 "날짜를 클릭하면 그 날의 출결, 시험, 일지, 메모를 확인할 수 있어요" 안내
  - 하단에 메모 입력: 드롭다운(카테고리 선택) + 입력 + 저장 (한 줄 정리)
- 오른쪽 달력 그리드:
  - 월 네비게이션 (교육 기간 내 월만)
  - 일~토 7열 표준 달력
  - 하단 dot: 출석(초록), 지각(주황), 결석(빨강), 일지(보라), 시험(파랑)
  - 시험 점수 숫자 표시 (80↑ 초록, 60↑ 주황, 60↓ 빨강)
  - 교육 기간 밖/미래 날짜 흐리게 처리
- 기존 독립 MemoSection 제거 → CalendarWithMemo 컴포넌트로 통합

### 3. 상세 페이지 2열 레이아웃 재배치 (4:6 비율)
- **왼쪽 (40%)**:
  - HR 조언 카드 (하락징후형/복합위기형 등 — dim 배경 별도 카드)
  - 적응지수 산정 근거 (레이더 차트 + 8항목 테이블 + 부정 신호 감점)
  - 출결 이력 (날짜/출근/퇴근/상태 테이블)
- **오른쪽 (60%)**:
  - 교육 캘린더 + 메모 통합
  - 차시별 점수 추이 (본인 vs 반 평균 바 차트)
  - 카테고리별 학습 현황 및 취약 영역
  - 교육일지 + 실습일지
  - 질문 이력
- 프로필 카드에서 HR 조언 제거 (왼쪽 컬럼에 별도 카드로 분리)

### 4. 취약 영역 카테고리별 학습 현황에 통합
- 기존 별도 "취약 부분" 카드 제거
- "카테고리별 학습 현황" → "카테고리별 학습 현황 및 취약 영역"으로 제목 변경
- 60% 미만 태그에 빨간 배경(`var(--red-dim)`) + "← 취약" 표시 + 굵은 글씨
- 정답률 퍼센트 + 정답/전체 문항 수 동시 표시

### 5. 메뉴명 변경
- 사이드바: "개별분석" → "교육생 분석"
- 페이지 제목: "개별분석" → "교육생 분석"
- 질문 페이지 링크: "개별분석 보기 →" → "교육생 분석 →"

### 6. 시행착오 및 제거된 기능
- "이번 주 챙길 교육생" 액션 대시보드: 구현 후 제거 — 홈 대시보드 "주의 교육생"과 중복, 단순 점수 하락만으로 표시하기엔 과함
- 목록형 타임라인: 구현 후 제거 — 날짜별 리스트 형태로는 패턴 인지가 어려움, 달력 형태로 전환
- 5열(월~금) 주 단위 캘린더: 구현 후 제거 — 홈 달력과 UI 통일성 부족, 7열 표준 달력으로 전환
