### 2. /my 홈 전면 재구성 (관리자 홈 스타일)

**기존**: 프로필 카드 + 학습 피드백 + 차시별 점수 + 오답 모아보기 (단순)
**신규**: 캘린더 + 체크리스트 + 스탯 + 취약영역 + 점수차트 + 일지/실습일지 + 오답 (dashboard 스타일)

**최종 레이아웃**:
1. 인사 섹션 (`안녕하세요, 곽현서님 · 입문교육 24일차 D-7`)
2. **[2fr + 1fr] 캘린더 + 오늘 할 일/스탯**
   - 왼쪽: 수업 캘린더 (미니 월간 + 선택일 수업 테이블)
   - 오른쪽: 오늘 체크리스트 (출근/교육일지/실습일지/공지) + 개인 할 일 + 내 상태 스탯 (평균/출석률/일지 제출률)
3. **[1fr + 2fr] 공부 필요 영역 + 차시별 점수 추이**
4. **[1fr + 1fr] 교육일지 + 실습일지** (각 4개 카드, 2×2 그리드, "오늘 일지 쓰기 →" 액션 링크)
5. 오답 모아보기 (테이블 형식, 차시별로 응시/오답/정답률 + 자세히 펼침)

### 3. 수업 시간표 기능 (LESSONS 데이터)

**CSV → TS 데이터 변환 파이프라인**:
- 노션에서 export한 CSV 파싱 (`프로젝트_기록/ExportBlock-2181aa61-.../교육 시간표.csv`)
- 노션 CSV의 timezone 라벨 (UTC/GMT+9 혼재) **무시**, 한글 날짜/시간 값만 그대로 추출 → KST로 처리
- `app/my/lessons-data.ts` 자동 생성 (23일치 수업)
- 종일 이벤트 지원 (안성공장 견학 등)

**LessonCalendar 컴포넌트**:
- 파란 배경 미니 월간 캘린더 (왼쪽, 360px 고정) + 선택한 날짜의 수업 테이블 (오른쪽)
- 월간 이동 (‹ / ›), 오늘/선택일 시각 구분, 수업 있는 날에 점 표시
- 테이블 컬럼: 시간 / 내용 / 강사 (장소는 요청으로 제거)
- 카드 높이 420px 고정 → 수업 많아도 내부 스크롤만 발생

**수동 커스터마이징**:
- 매장실습 4일 (4/11, 4/12, 4/18, 4/19) 각 날짜에 2개 매장 병기 (노원+송파 / 논현+분당서현)
- 시간 `10:30 ~ 19:30`으로 통일

### 4. 학생 개인 할 일 (student_todos) 기능 신규

**DB**: `student_todos` 테이블 (`supabase/student_todos.sql`)
- 컬럼: id, student_id(FK), date, text, done, sort_order, timestamps
- RLS 활성화 + 앱 레벨 제어

**API**: `/api/student-todos` (GET/POST/PATCH/DELETE)
- GET `?student_id=X&date=YYYY-MM-DD`
- POST 할 일 추가 → 학생 id + 날짜 + 텍스트
- PATCH 완료 토글 / 텍스트 수정
- DELETE 삭제

**UI**: 홈 "오늘 할 일" 카드 안에 "내 할 일" 섹션
- 체크박스 클릭 → 완료 토글 (완료 시 취소선 + muted)
- × 버튼 → 삭제
- Enter로 빠르게 추가
- 데스크톱 420px 고정 / 모바일 `height: auto`로 자유롭게 늘어남
