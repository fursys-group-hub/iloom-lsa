# 2026-04-13 — 디자인 시스템 전면 리뉴얼

> awesome-design-md 레포의 Notion DESIGN.md를 참고하여, 입문교육 사이트 전체의 디자인 토큰과 UI 패턴을 체계적으로 정리한 작업.

---

## ✅ 오늘 한 것:

### 1. globals.css 디자인 토큰 리뉴얼
- **라이트모드를 기본값으로 전환** (`:root` = 라이트, `body.dark` = 다크)
- ThemeToggle.tsx: `body.light` 토글 → `body.dark` 토글로 변경
- 배경: `#F7F8FA` (tint 배경) + `#FFFFFF` (흰 카드) → 카드 구분감 확보
- 테두리: `rgba(0,0,0,0.10)` "속삭임" 스타일로 변경
- 그림자: 단층 → **4겹 다층 자연광 그림자** (`--shadow-sm`, `--shadow-md`)
- 타이포: h1~h4에 **마이너스 letter-spacing** 추가 (-0.025em ~ -0.01em)
- body line-height: 1.6 → 1.5로 조정
- 색상: 원래 액센트 색상 유지 (초록/주황/빨강/보라 등)

### 2. 전체 13페이지 카드/뱃지/그림자 통일
- **카드**: `padding: '20px 24px'` + `boxShadow: var(--shadow-sm)` 일괄 적용
- **뱃지(pill)**: `fontSize: 12, fontWeight: 600, padding: '3px 10px'` 통일 (기존 10~13px 혼재)
- **borderRadius 하드코딩 제거**: `4` → `var(--radius-xs)`, `8` → `var(--radius-sm)` 등 CSS 변수로 교체
- 적용 파일: DashboardClient, attendance, tests, education-logs, practice, questions, announcements, students, reports, overview, analytics, settings, evaluations

### 3. 대시보드 홈 정리
- D-day 뱃지 + "입문교육 진행중" 상태 뱃지 **중복 제거** (D-day만 유지)
- 타임라인 바 radius 통일 (`radius-sm`)
- "전체보기 →" 링크 통일 (`cardLinkStyle`: fontSize 12, fontWeight 400, text-muted)
- 주의 교육생 뱃지: `badgeBase` 공통 스타일로 통일
- **숫자 카드 라벨 회색 통일**: 라벨은 항상 `--text-tertiary`, 숫자만 색상 적용 (심화교육 패턴)

### 4. 탭 디자인 — 밑줄 탭으로 전면 교체
- 기존: 세그먼트 컨트롤 (파란 배경 토글 버튼) → **밑줄 탭**으로 변경
- 탭 위치: 제목 옆 → **제목 아래**로 이동
- 첫 탭 왼쪽 패딩 `0px`로 제목과 정렬 맞춤
- 활성 탭: 파란 밑줄 2px + fontWeight 600
- 비활성 탭: text-muted + fontWeight 400
- 중복 제목 제거 (탭이 "차시별 성적"이면 아래 "차시별 성적 현황" 삭제)
- 적용 4곳: 테스트(2개), 개별분석, 심화교육
- 질문관리 필터(전체/답변 대기/답변 완료/보관됨) → 밑줄 탭으로 변환
- 교육일지 필터(전체/우수/미완료) → 밑줄 탭으로 변환

### 5. 아바타 32px 통일
- 기존: 24~40px 혼재 → **32x32px** 통일 (목록/테이블용)
- 색상: `background: var(--blue-dim), color: var(--blue)` 통일
- 적용: DashboardClient, attendance, education-logs, students, questions, tests, overview
- 예외: 프로필 상세(56px), 심화교육 상세(52px) 유지
- 출결 테이블 + 심화교육 테이블에 아바타 신규 추가

### 6. 드롭다운(select) 전체 통일
- `padding: '8px 14px', borderRadius: var(--radius-sm), fontSize: 14, fontWeight: 600`
- 적용 10개 파일, 16개 select 요소
- 예외: 테이블 셀 편집용 select는 `6px 12px` 유지

### 7. 테이블 셀 패딩 통일
- th/td 전부 `padding: '12px 16px'`로 통일
- 기존: 출결(14px 16px), 테스트(8px 12px), 개별분석(6px 8px) 혼재

### 8. 필터 버튼 통일
- 활성: `background: var(--blue), color: #fff`
- 비활성: `background: transparent, border: 1px solid var(--border)`
- 적용: 개별분석, 교육일지, 심화교육

### 9. 사이드바 접기/펼치기
- 그룹 라벨(일일 운영, 성과 분석 등) 클릭 시 하위 메뉴 접기/펼치기
- ▼ 화살표 회전 애니메이션
- 현재 보고 있는 페이지가 속한 그룹은 접어도 활성 메뉴 표시

### 10. 보조 버튼 테두리 제거
- "수정", "취소" 같은 보조 액션 버튼: `border: none` → 텍스트만
- 출결 페이지 `smallBtnStyle`에 적용

### 11. DESIGN_SYSTEM.md 문서 작성
- 18개 섹션으로 전체 디자인 규칙 문서화
- CLAUDE.md에 `@DESIGN_SYSTEM.md` 참조 추가
- Anti-patterns 목록 포함 (하지 말 것)

## 🤔 결정한 것:
- **색상은 원래대로 유지**: Notion 스타일 단색 시도 → 너무 안 보여서 원래 다색 뱃지로 복원
- **라벨은 회색, 숫자만 색상**: 심화교육 패턴이 가독성 좋아서 채택
- **탭은 밑줄 스타일**: 세그먼트 컨트롤(iOS) 시도 → 탭처럼 안 느껴져서 밑줄로 최종 결정
- **아바타 기본 크기 32px**: 28px은 너무 작고, 40px은 너무 커서 32px 채택
- **테이블 패딩 12px 16px**: 출결 기준으로 통일 (너무 빽빽하지도, 넓지도 않게)
