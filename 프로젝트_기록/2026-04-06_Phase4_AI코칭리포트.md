### 5. 구조화된 렌더링
- 마크다운 테이블(`| col |`) → HTML `<table>` 자동 변환
- 반복 오답 → 카드형 (문제 + 보기 + 학생답 + 정답 + 해설)
- 시험 성적 → 차시별 바 차트 (반 평균 대비 +/- 색상 코딩)
- `**볼드**` → `<strong>`, `==형광펜==` → `<mark>` 인라인 변환
- 첫 주 교육 추천 → 테이블 형식

### 6. 리포트 내용 설계
- **왼쪽 (핵심)**: 피드백 + 시험 성적 차트 + 추천 교육 스타일 + 첫 주 교육 추천
- **오른쪽 (데이터)**: 교육 과정 체크리스트(O/△/X + 자립도) + 반복 오답 문항
- 중간 세로 구분선

### 7. 교육자 메모 전문 리라이팅 (Claude Code 직접)
- 12건 메모를 생활기록부 톤으로 리라이트 (`scripts/memo-rewrites.json`)
- "소심함" → "신중하고 조심스러운 성향, 격려와 연습 기회 제공이 효과적"
- "컨디션 기복" → "컨디션에 따른 학습 집중도 편차가 관찰됨"
- "시험 1등 쾌거" → "시험 1등을 달성하는 성과를 거두며 노력이 결실을 맺는 모습이 인상적"
- 구체적 관찰 내용(SCT 교육 주도 등)은 원문 유지, 표현만 다듬기

### 8. 시험 성적 분석 개선
- ~~단순 점수 비교~~ → **반 평균 대비 상대 성적** 분석
- 차시별 과목이 다르므로 "95→72 하락=복습 부족"은 잘못된 분석
- "1차시(+23) 6차시(+30) 9차시(+10)" 형태로 반 평균 대비 gap 표시

### 9. 학생 상세 페이지 연동

- tag_tracking 극복/고질적 태그 뱃지 표시
