# 2026-04-13 — 매장 실습 보고서 기능 추가

## ✅ 오늘 한 것:

### 1. 실습일지 보고서 생성 기능
- 실습일지 페이지에 **"📋 보고서 프롬프트 복사"** 버튼 추가
- 클릭하면 해당 날짜의 실습일지 데이터(상담/견적/수주/금액 + 4개 섹션 텍스트)를 자동 집계
- 집계된 데이터를 AI 코칭 피드백 생성용 프롬프트로 변환하여 클립보드에 복사
- Claude Code에 붙여넣으면 전문적인 코칭 피드백 작성 + DB 저장까지 완료

### 2. `/api/practice-report` API
- GET: 날짜별 실습일지 데이터 집계 + 기존 보고서 존재 여부 확인
- Supabase에서 `실습일지` 태그 노트만 필터링 → 학생별 실적·내용 파싱

### 3. 리포트 페이지 practice 타입 지원
- `coaching_reports` 테이블에 `report_type: 'practice'` 저장
- 초록색 "매장 실습 보고서" 뱃지로 구분
- `PracticeReportView` 전용 렌더러 — 일자별 통합 보고서 (14명 한 화면)
  - `📌 **이름**` 헤더, `→` 피드백 들여쓰기, `**볼드**` 강조, `---` 구분선
- 기존 수정/PDF 기능 그대로 활용 가능
