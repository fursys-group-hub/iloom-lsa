# 2026-04-01 — Phase 3: 공지사항 + 코멘트 + 모바일 반응형

## ✅ 오늘 한 것:

### 1. 기수관리 페이지 — 퇴사 처리 + 비밀번호 초기화
- 교육생 테이블에 **🔑 비밀번호 초기화** 버튼 추가 (0000으로 리셋)
- **퇴사 처리** 버튼 + 모달 (퇴사일/사유 입력) 추가
- 퇴사자: 반투명(opacity 0.5) + 이름 취소선 + 빨간 "퇴사 MM-DD" 뱃지
- **복구** 버튼으로 퇴사 취소 가능
- `StudentRow` 인터페이스에 `is_dropped`, `dropped_at`, `drop_reason` 추가

### 2. 교육일지 퇴사자 회색 처리
- 퇴사한 학생의 노트 카드: opacity 0.4로 회색 처리
- 이름에 취소선 + 빨간 "퇴사" 뱃지 표시
- 내용은 펼쳐서 볼 수 있음 (삭제 아님)

### 3. 사이드바 영업지원 챗봇 링크
- 관리자 사이드바 + 교육생 사이드바에 💬 **영업지원 챗봇** 링크 추가
- `https://iloom-saleschatbot.vercel.app/` 새 탭으로 열림

### 4. 📢 공지사항 기능 (기수별 관리자→교육생)
- **DB**: `announcements` 테이블 (batch_id, title, content, priority)
- **API**: `/api/announcements` CRUD (GET/POST/PATCH/DELETE)
- **관리자 페이지** (`/dashboard/announcements`):
  - 사이드바에 📢 공지사항 메뉴
  - 기수별 공지 작성/수정/삭제
  - 우선순위 3단계: 📢 일반 / ⚠️ 중요 / 🚨 긴급
  - 왼쪽 컬러 라인으로 우선순위 구분
- **교육생 로그인 팝업**:
  - 안 본 새 공지 → 팝업으로 표시
  - 여러 개면 이전/다음 네비게이션
  - `localStorage`에 마지막 확인 시간 저장
- **교육생 모아보기** (`/my/announcements`):
  - 사이드바에 📢 공지사항 메뉴
  - 아코디언 펼침식, 24시간 이내 NEW 뱃지
- **auth API**: 학생 응답에 `batchId` 추가 (기수별 공지 필터링)

### 5. 💬 교육일지 코멘트 기능 (관리자↔교육생 대화)
- **DB**: `note_comments` 테이블 (note_id, author_role, author_name, content)
- **API**: `/api/note-comments` (GET/POST/DELETE, note_ids 일괄 조회 지원)
- **관리자 교육일지**:
  - 노트 펼치면 💬 코멘트 입력란 표시
  - iMessage 스타일 말풍선 (관리자: 파란색 우측, 교육생: 회색 좌측)
  - 자기 코멘트 hover 시 × 삭제 버튼
  - 코멘트 추가/삭제 시 💬 뱃지 카운트 실시간 반영
- **교육생 교육일지**:
  - 관리자 코멘트가 있을 때만 코멘트 영역 표시 (없으면 숨김)
  - 답글 입력 가능 (파란색 말풍선, 우측 정렬)
  - 노트 카드에 💬 코멘트 수 뱃지 표시
