# 2026-03-28 — 학생 개인 페이지 + AI 인사이트 + 로그인 개선

## ✅ 오늘 한 것

### 1. 학생 개인 페이지 (`/my`)
- 사이드바: 홈 / 출결 / 테스트 / 교육일지 / 실습일지(준비중) / 질문하기(준비중)
- `/my` 홈: 프로필 + 학습 피드백(약점/주의/강점 태그) + 차시별 점수 라인 차트(반 평균 비교) + 오답 모아보기
- `/my/tests`: 차시별 내 점수 카드 + 문항별 O/X + 보기 + 답안 비교 + 해설 (다른 학생 것 안 보임, 문제은행 안 보임)
- `/my/attendance`: 내 출결 기록 (출근/지각 요약)
- `/my/notes`: 교육일지 작성 (오늘 한마디 + 자신감 체크(😎😊🤔😵) + 태그 + 핵심 필기)
- 모바일 반응형 (사이드바 숨김 + 햄버거 메뉴)

### 2. 로그인 개선
- 관리자: 이름 `김수지` + 비밀번호 `1230` → `/dashboard`
- 학생: 이름 + 비밀번호(숫자 4자리) → `/my`
- `/api/auth` API로 서버에서 인증 (기존 클라이언트 하드코딩 제거)
- 기수 관리에서 학생별 비밀번호 설정/확인 가능 (기본 `0000`)

### 3. AI 교육 인사이트 (Gemini 2.5 Flash)
- 홈 화면에 "🤖 AI 교육 인사이트" 카드
- "새 분석 생성" 버튼 → Gemini가 오답 데이터 분석
- 프롬프트: 일룸 교육 전문가 페르소나, 취약 영역 분석 + 교육 추천 + 변형 문제 제안
- 인사말 없이 구조화된 리포트만 출력
- `teaching_insights` 테이블에 저장 → 다음에도 열람 가능

### 4. 교육생 상세 페이지 개선
- 차시별 점수 추이: 반 평균 라인 추가 (파란선: 학생, 회색선: 반 평균)
- 차트 다크모드 적용
- 상태 판별 기준 토글 (필터 우측 드롭다운)

### 5. 문제은행 개선
- 해설 수정 기능 추가 (정답 수정 시 해설도 함께)
- 보기 표시 줄바꿈 처리 (파이프 → 개행 구분)

### 6. 기타
