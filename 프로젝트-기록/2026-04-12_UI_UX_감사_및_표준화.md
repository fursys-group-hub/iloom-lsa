# 2026-04-12 — UI/UX 전체 감사 및 표준화

## ✅ 오늘 한 것:

### 1. 실습일지 새벽 5시 보정 + 날짜 선택 드롭다운
- 새벽 5시 이전 작성 시 전날로 자동 보정 (교육일지와 동일)
- 실습일 날짜 드롭다운 추가 (수동 날짜 변경 가능)
- 저장 시 `target_date` 전달 → 선택한 날짜 정오(KST)로 기록
- 학생 + 관리자 페이지 모두 적용

### 2. 실습일지 카드 색상 통일
- 초록색(green-dim) → 교육일지와 같은 흰색/기본 테마로 전환
- 카드, 폼, 상세보기, 새 실습일지 버튼 전부 적용
- 학생 + 관리자 양쪽 모두

### 3. 전체 페이지 UI 표준화 (1차)
- 페이지 제목: 학생홈 26px, 리포트 24px → **전체 28px/700**
- 빈 상태: padding 48~64, emoji 40~48 → **padding 48, emoji 40, 제목 16**
- 테이블 헤더: 출결 10px 14px → **12px 16px**
- 모달 padding: 32px → **28px**
- 카드 padding: 리포트 20px 24px → **24**
- 뱃지: 실습일지 학생 11px → **12px, 2px 8-10px**

### 4. 날짜 선택 pill → 드롭다운 전환
- 출결, 교육일지, 실습일지 — 가로 pill 버튼 → `<select>` 드롭다운
- 공지사항 기수 선택 pill → 드롭다운
- 탭(뷰 전환/필터)과 드롭다운(날짜/기수 선택) 역할 명확히 분리

### 5. 필터/드롭다운/뱃지 크기 통일
- 필터 pill: 질문관리 7px 16px → **6px 14px, radius-sm**
- 드롭다운: 학생 필터 6px 12px → **8px 14px, fontSize 14**
- 상태 뱃지: 질문 11px → **12px, 2px 10px**

### 6. 액션 버튼 + 교육효과 헤더 통일
- 테스트 "새 응답만 추가" 초록 테두리 → ghost 스타일
- 교육효과: h1 22px/800 → **h2 28px/700**, 자체 padding 제거

### 7. 헤더 구조 통일
- 서브타이틀 제거 (공지사항, 기수관리만 있던 것)
- outer gap: 테스트 32px, 교육생 32px → **전체 24px**
- 질문관리: marginBottom → flex column gap 24 구조로 변경

### 8. 사이드바 그룹핑 + 이름 변경
- 12개 나열 → 5개 그룹으로 정리
- "교육생" → "개별분석"으로 변경
- 그룹: 홈 / 일일 운영 / 성과 분석 / 심화교육 / 설정

## 🤔 결정한 것:
- 날짜 선택 = 드롭다운, 뷰 전환/필터 = pill 버튼 (역할 분리)
- 서브타이틀은 일관성을 위해 전체 제거
- 사이드바 그룹: 일일 운영(출결/테스트/교육일지/실습일지/질문) / 성과 분석(개별분석/리포트/교육효과) / 심화교육 / 설정(공지/기수관리)
