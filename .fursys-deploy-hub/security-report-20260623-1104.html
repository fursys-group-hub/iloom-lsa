평문 비밀번호 암호화 (app/api/auth/route.ts)

아래 글을 그대로 복사해 AI 도구에 붙여넣으면 고쳐줍니다.

app/api/auth/route.ts 파일을 열어보면 매장 관리자(manager)와 교육생(student)의 비밀번호를 DB에서 꺼내 평문으로 바로 비교하고 있어요.

개선 방법:
1. bcryptjs 패키지를 설치해주세요 (npm install bcryptjs)
2. 비밀번호 비교 시 bcrypt.compare(입력비밀번호, DB해시값) 를 사용하세요
3. 기존 평문 비밀번호는 마이그레이션 스크립트로 해시로 변환해야 해요

단, 이 앱이 사내 내부 교육 관리 앱임을 감안해 우선순위는 중간이에요.
