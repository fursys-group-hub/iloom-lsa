# 2026-05-04 — 키큰옷장 한샘 비교 자료 v2-3 정리

입주박람회용 컬렉트 키큰옷장 vs 한샘 비교 가이드(`키큰옷장_한샘비교_v2-3.html`)의 내용 검증, 메시지 일관성 통일, 법적 리스크 익명화, 색감 조정까지 한 번에 진행.

## ✅ 오늘 한 것

### 1. 데이터 정합성 검증 (연경님 전달본 vs HTML)

연경님 전달본 자료(한샘 베이직.png / 한샘 시그니처.png / 가격 비교.png)와 HTML v2-3 전체 대조하여 사실 정확도 확인.

**가격표 (8개 행 A~H)**: 모두 정확 일치 ✅
**스펙 비교**: 대부분 정확하나 3가지 부정확 발견 → 수정

### 2. 핵심 메시지 통일

방향성: **"한샘 가성비 라인 가격에 프리미엄 라인 내구성"**

| 위치 | 변경 전 | 변경 후 |
|---|---|---|
| 표지 (Page 1) | "한샘 프리미엄과 같은 견적가로, 한 단계 위 사양과 명품 부품까지" | "**한샘 가성비와 같은 가격에, 프리미엄급 내구성과 명품 부품까지**" |

→ 표지와 본문 메시지가 일관되게 정리됨

### 3. 한샘 베이직 서랍 레일 표기 정확화

원본 자료에 한샘 베이직 서랍재는 "**15mm PB + 코팅페이퍼 + 0.6mm ABS 에지 + 댐핑 언더레일**"로 명시되어 있는데, HTML에는 "미명시"로 표기되어 영업현장에서 반박 가능 → 정확화

| 위치 | 변경 전 | 변경 후 |
|---|---|---|
| Page 2 핵심 사양 한눈에 비교표 | 한샘 베이직 서랍 레일: "미명시" | "**댐핑 언더레일 (브랜드 미명시)**" |

(Page 5 Hettich 비교표는 "부품 브랜드"가 키이므로 "미명시" 그대로 유지 — 정확함)

### 4. Q4 응대 시나리오 사실 정정 (Page 10)

원본 검증 결과 한샘 시그니처는 **PP(우드) + LPM(레더) + 코팅페이퍼(레더) 3종 혼용**이었으나, HTML은 "PP우드 + LPM 혼용"으로 부정확 + "경계 부분 결합력 약점"이라는 추측성 비방 표현 사용.

| 블록 | 변경 |
|---|---|
| 핵심 메시지 | "PP우드는 일부 부위만, LPM은 전 부위 일체 — 결합 약점 없음" → "**한샘 시그니처는 PP우드 + LPM + 코팅페이퍼 3종 혼용, 컬렉트는 전 부위 LPM 일체**" |
| 응대 스크립트 | "경계 부분 결합력 약점" 추측 표현 삭제 → 사실 기반("뒷판·서랍재는 여전히 코팅페이퍼")으로 정정 |
| 핵심 숫자·근거 | "PP우드 + LPM 혼용" → "**3종 혼용**" + "한샘 시그니처 뒷판·서랍재: 여전히 코팅페이퍼" 항목 추가 |

### 5. Page 2, 3 비교표에 "뒷판·서랍재 마감" 행 추가 (옵션 B)

Q4 멘트에서 "코팅페이퍼" 사실이 정정됐는데 정작 비교표에는 빠져있어서 일관성 부족.

→ 두 표에 "**뒷판·서랍재 마감**" 행 추가:
- 한샘 베이직: 코팅페이퍼
- 한샘 시그니처: 코팅페이퍼(레더) ← 시그니처도 결국 베이직과 동일하다는 시각적 임팩트
- 컬렉트: **LPM 일체**

### 6. 행 간격 조절 (페이지 넘침 해결)

행 추가로 Page 2, 3이 페이지를 넘어가는 문제 해결:

- thead 헤더 padding 10px → 8px

### 7. 표지 목차(CONTENTS) 추가

### 9. 🚨 법적 리스크 익명화 (가장 중요한 변경)

**배경**: 비교광고법상 "한샘"이라는 실명 명시는 부당비교광고/비방광고로 해석될 위험. 한샘은 법무팀이 강력해서 모니터링 가능성 높음. 인쇄물에 박힌 텍스트는 증거로 남음.

**방식**: 부분 익명화 — 고객 노출 영역만 익명화, 영업 내부용은 실명 유지

| 영역 | 처리 |
|---|---|
| **Page 1~8 (고객 배포용)** | "한샘" / "리하우스" → **"타사"** 일괄 변경 |
| **Page 9~10 (영업 내부 교육용)** | 한샘 실명 그대로 유지 (응대 멘트는 실명이 자연스러움 + 이미 "고객 비공개" 표시 있음) |

**매핑**:
- 한샘 베이직 → 타사 가성비 라인
- 한샘 시그니처 → 타사 프리미엄 라인
- 한샘 vs 컬렉트 → 타사 vs 컬렉트
- 리하우스 가성비/프리미엄 라인 → 타사 가성비/프리미엄 라인
- 입주박람회 프리미엄 라인 → 타사 프리미엄 라인 (통일)
- 한샘이 명시조차 안 하는 → 타사가 명시조차 안 하는

총 25개 위치 수정.

### 10. 색감 조정 (옵션 A + B)

수지님이 모니터로 검토 시 흑백 미니멀 톤이 눈에 부담된다고 피드백 → 인쇄물 가독성은 유지하면서 톤 다운.

**옵션 A — 검정 톤 다운**:
- 모든 `#1A1A1A` → `#2A2A2E` (일괄 변경)
- 본문 텍스트, 강조 보더, 헤더 강조선 등 모든 검정
- 인쇄물에서는 차이 거의 없음, 모니터에서 부드러워짐

**옵션 B — 컬렉트 컬럼 그레이지 톤**:
- 차가운 회색 #F5F5F5 → 그레이지 #F7F5F1 (회색빛 살짝 도는 베이지)
- 차가움 → 따뜻한 가구 브랜드 톤
- 톤 결정 과정: #F8F4ED (베이지) → #FCF9F2 (옅은 베이지) → #F2EFE9 (그레이지) → **#F7F5F1 (옅은 그레이지)** 최종 결정

## 🤔 결정한 것

### 자료 운영 방식
- **한 파일로 운영**, 영업이 고객 응대 시 1~8장만 보여주고 9~10장은 본인 교육용으로 활용
- 굳이 고객용/내부용 파일 분리 안 함
- Page 9~10에는 "고객 비공개" 명시 + 부록 처리 + 표지 목차에서도 회색 톤으로 흐릿하게 표시

### 법적 리스크 대응
- **부분 익명화 방식 채택**: 고객 노출 영역(Page 1~8)만 익명화
- "한샘" → "타사 가성비/프리미엄 라인" 일관 매핑
- 영업이 구두로 "한샘"이라 말하는 건 OK (인쇄물에 증거 안 남음)

### 색감 톤
- 흑백 미니멀 기조는 유지 (프리미엄 메시지에 부합)
- 본문 검정만 살짝 부드럽게 (#2A2A2E)
- 컬렉트 컬럼은 그레이지 (#F7F5F1)로 가구 브랜드 톤 살짝 살림
- 이외 회색 영역(가격표 헤더, compact-tag 등)은 차가운 회색 그대로
