# 🛋️ 키큰옷장 입주박람회 키오스크 (쇼룸 웹)

기존 인쇄용 PDF 자료(`키큰옷장_한샘비교_v2-3.html`)를 가로형 TV 키오스크에서 손님이 직접 만지면서 볼 수 있는 **인터랙티브 웹 화면**으로 변환한 자료예요.

> "계속 틀어놔도 영상처럼 보이고, 손님이 만지면 인터랙션도 되는" 박람회 부스용 키오스크 자료

---

## 🎯 목적

- 박람회 부스에 가로형 TV(1920×1080) 켜놓고 **손님이 직접 만지면서 컬렉트 키큰옷장의 강점을 체험**할 수 있게 만들기
- 영업사원이 옆에서 설명하지 않아도 손님이 핵심 메시지를 자동으로 받게 하기 (Attract 모드)
- 손님이 관심 보이면 부품을 확대해서 보고, 가격 직접 비교해보고, 옷장 구성도 짜볼 수 있게 하기 (Interactive 모드)

---

## 🖥️ 동작 방식 (키오스크)

### Attract 모드 (Auto-play)
- 손님이 안 만질 때 — **8초마다 슬라이드 자동 전환** + 부드러운 페이드 효과로 영상처럼 보임
- 부스 옆 지나가는 손님 시선 끌어들이기 용도

### Interactive 모드 (Touch)
- 손님이 화면을 터치하면 자동재생 멈춤
- 좌우 화살표 / 스와이프 / 하단 점 클릭으로 페이지 이동
- 부품 사진/모듈 클릭으로 상세 보기
- 가격 슬라이더로 구성 바꿔가며 가격 비교

### 자동 복귀
- 30초 동안 손님이 안 만지면 다시 표지로 돌아가서 자동재생 시작

## 📋 화면 구성 (8개 슬라이드)

| # | 슬라이드 | 주요 내용 | 인터랙션 |
|---|---|---|---|
| 1 | **표지 (인트로)** | "타사 가성비와 같은 가격에, 프리미엄급 내구성과 명품 부품까지" | 시작 버튼 (Attract 모드 빠져나옴) |
| 2 | **한눈에 비교** | 7개 핵심 사양 비교표 + 350원 차이 메시지 | 표 행 호버 강조 |
| 3 | **표면재 차이** | LPM vs 코팅페이퍼 (Part 1) | **사진 클릭 → 확대 모달**, 단면 이미지 비교 |
| 4 | **두께 차이** | 윗판 18.5mm / 옆판 15.5mm | 0.5mm 차이 시각화 (애니메이션 바) |
| 5 | **부품의 차이** | Hettich, Salice, 알루미늄 옷걸이봉 (Part 2) | **부품 카드 클릭 → 확대 모달** + 50,000회 / 200,000회 강조 |
| 6 | **모듈 + 시뮬레이터** | 모듈 6종 + 손님이 직접 옷장 구성 짜기 | **모듈 클릭하면 옷장에 추가/삭제, 총 폭 자동 계산** |
| 7 | **도어 옵션** | 리디 도어 vs 스톤W 도어 | 좌우 토글로 비교 |
| 8 | **가격 비교 슬라이더** | 8개 구성 (3통반/3통/2통/1통반 × 붙박이장/옷장) | **좌우 화살표로 구성 변경 → 한샘 vs 컬렉트 가격 즉시 비교** |

## 🎨 디자인 컨셉

기존 인쇄용 자료 톤 유지:
- **폰트**: Paybooc (이미 폴더에 있음)
- **메인 컬러**: 검정 #2A2A2E, 컬렉트 그레이지 #F7F5F1
- **포인트**: 한샘 가성비/프리미엄 vs 컬렉트 3단 비교 컬럼 톤
- **영상감**: 슬라이드 전환 시 부드러운 페이드, 슬라이드 안에서도 콘텐츠가 페이드인 (시선 유도)
- **터치 친화적**: 버튼 최소 60×60px, 폰트 최소 18px (TV 멀리서 봐도 보이게)

## ⚙️ 사용법

### 박람회 현장에서
1. TV에 노트북 연결 (HDMI)
2. 노트북에서 `index.html` 더블클릭 → 브라우저로 열림
3. **F11 누르기 (전체화면)** — 브라우저 UI 사라지고 풀스크린 키오스크 화면
4. 그대로 두면 자동재생, 손님이 터치하면 인터랙션

### 키보드 단축키 (디버그용)
- **F11**: 전체화면 토글
- **←/→**: 슬라이드 이동
- **Space**: 자동재생 일시정지/재개
- **Esc**: 모달 닫기
